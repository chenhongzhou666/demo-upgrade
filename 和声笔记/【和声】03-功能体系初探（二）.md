## 二、新概念

### 正三和弦

**I、IV、V 级（TSD）三个和弦统称为「正三和弦」**，因为它们最重要、最「正」。

对应的，**ii、iii、vi、vii°** 四个和弦统称为「**副三和弦**」——现在不用记，以后讲到再说。

### 连接规则

TSD 之间的连接关系：

```
T ←→ D   ✅ 正格进行
T ←→ S   ✅ 变格进行
S → D    ✅ 情绪递进（从刚出门到玩尽兴）
D → S    ❌ 反功能进行（初期禁止！）
```

**D → S 为什么被禁止？**

1. **感性上违和**：玩尽兴了想回家 → 突然被告知才刚出门，情绪倒退而非递进
2. **阻碍属功能的使命**：D 唯一想做的事就是回家（D→T），D→S 打断这个目的
3. **模糊调性**：缺少 D→T 就无法有效建立/巩固调性，听众搞不清是什么调

举例：C大调写 T → D → S，缺少 D→T，听上去反而更像 F 大调的 T→S→D→T。

> ⚠️ D→S 并非永远不能用！以后会讲具体运用条件（比如你就是要模糊调性）。但**学习前期必须避免**。

### 和声进行的术语

| 术语 | 含义 | 包含功能 |
|------|------|---------|
| **变格进行** | T 与 S 之间的连接 | T + S |
| **正格进行** | T 与 D 之间的连接 | T + D |
| **完全进行** | T → S → D → T 全部包含 | T + S + D |

更细致的分类（知道就行，不必死记）：

| 细分 | 方向 |
|------|------|
| 半正格进行 | T → D |
| 正格进行/正格终止 | D → T（在结尾处叫终止） |
| 半变格进行 | T → S |
| 变格进行/变格终止 | S → T（在结尾处叫终止） |
| 补充终止 | D→T 之后再加 S→T |

## 三、TSDT 为什么重要

### 一层：情绪推进最完整

```
T → S → D → T
在家 → 出门 → 玩尽兴 → 回家
稳定 → 逐渐紧张 → 最紧张 → 解决
```

比单独的变格（T↔S）或正格（T↔D）更完善。

### 二层：展示全部调内音

| 和弦 | 组成音（级数） |
|------|--------------|
| T（I） | ① ③ ⑤ |
| S（IV） | ④ ⑥ ① |
| D（V） | ⑤ ⑦ ② |

**1234567 七个音全部出现在 TSD 三个和弦中！** 所以 T→S→D→T 跑一遍，就把整个调的音全展示了 → 建立了调性，再通过 D→T 巩固调性。

### 三层：流行音乐的基础

流行模板「**1451**」= T → S → D → T，就是完全进行的简化说法。

> 把 TSD 三个和弦用好了、理解透了，就算和声入门了。

## 四、哲学讨论：为什么 T 稳定、D 想回家？

西蒙用最后几分钟讨论了「凭什么？」这个深层问题。

### 解释一：泛音列

- 任何一个音的**前 6 个分音**天然构成大三和弦（如 C 的前 6 分音 = C C G C E G）
- 所以大调的主和弦（大三和弦）天然稳定
- 第 4-7 分音构成属七和弦 → 属功能有「第二属性」
- 与根音成**纯五度**的音总是最先出现 → 主属关系天然重要

**但这个解释的漏洞**：无法解释小调主和弦（小三和弦）为什么也稳定。

### 解释二：五度相生律

- 纯八度 = 1:2，次简单的整数比是 **2:3（= 纯五度）**
- 从某个音出发，通过 2:3 找到的**第一个新音**就是它的属音
- 这就是主属关系在数理上的起源

### 西蒙的结论

这些解释都是**侧面引证**，无法从本质上直接证明。更像是在「大小调和声体系」这个既定框架内，用侧面理论辅助理解。

**类比牛顿定律**：你问「为什么没外力就匀速直线运动」，牛顿也只能侧面解释。在宏观低速世界里，这就是需要接受的框架前提。

> 核心态度：**把框架当作工具去用，不必一直追问框架本身为什么是这样。** 几百年来西方大小调体系通过耳濡目染让全世界接受了这些设定，我们在这个框架内学习和声实操即可。

## 课后练习

和上节课一样：**任选几个大小调，写出它们的 T、S、D 分别是哪三个和弦**，用 TSD 参考表核对。

> **不要死记！** 知道怎么推导就好。表是用来核对的。

推导规则：
- **大调**：I、IV、V 级均构建大三和弦
- **小调**：i、iv 构建小三和弦，V 构建大三和弦（升 VII 级）
