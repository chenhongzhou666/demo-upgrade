## 核心概念

### 为什么是四个声部？

四声部是几百年实践得出的 **既饱满又简洁** 的配置：

| 对比 | 问题 |
|------|------|
| < 4 声部 | 三和弦/七和弦构成音表达不完整，需频繁省略音，不饱满 |
| = 4 声部 | 可完整表达三和弦（重复一个音）、七和弦（四个音各一声部），恰到好处 |
| > 4 声部 | 构成音重复过多 → 音响累赘，削弱各声部独立性 |

**现实对应**：
- 管弦乐队木管组/铜管组：常规 4 声部配置
- 弦乐组：看似 5 个（小提1/小提2/中提/大提/低音提），但低音提琴通常只是大提琴的低八度重复
- 混声合唱 SATB：Soprano / Alto / Tenor / Bass

> 学好四部和声后，少于 4 个（省略音）或多于 4 个（重复强调）都自然能处理。

### 四个声部的名称

```
高 ─ 高音声部 / 女高音(S) / 第一声部 ─┐
    │                                   ├── 外声部（人耳最敏感）
    │ 中音声部 / 女低音(A) / 第二声部 ─┐  │
    │                                   ├── 内声部（充实和声）
低 ─ 次中音声部 / 男高音(T) / 第三声部 ─┘  │
    │                                       │
    └── 低音声部 / 男低音(B) / 第四声部 ──── 外声部
```

- **高音声部** = 旋律声部（人耳听得最清楚）
- **低音声部** = 第二容易听清，重要性后续课程会体现
- **内声部**（中音+次中音）= 主要作用是充实和声

### 记谱规范

- 使用 **钢琴谱表**（高音谱号 + 低音谱号）
- 高音谱号：记 高声部 + 中声部
- 低音谱号：记 次中音声部 + 低声部
- **符干方向**：高声部、次中音声部符干朝上；中声部、低声部符干朝下
  - 这样能直观看出每个声部的横向进行

## 要点总结

1. 和声训练基于**四部和声框架**（SATB），这是科班标准路线
2. 四个声部从高到低：S（女高）→ A（女低）→ T（男高）→ B（男低）
3. 外声部 = S + B（人耳敏感），内声部 = A + T（充实和声）
4. 记谱用钢琴谱表，符干方向体现声部独立性
5. 目前只需记住概念，后续课程会教具体怎么写
