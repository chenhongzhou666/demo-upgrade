---

## 第一部分：变格补充终止

### 一、回顾：变格终止能不能用？

变格终止（S→T）**完全可以**直接用在全终止处，但：

| 终止类型 | 终止感 | 使用频率 |
|---------|--------|---------|
| 完全终止（S→D→T）/ 正格终止（D→T） | 强、有力 | ⭐⭐⭐ 绝对主流 |
| 变格终止（S→T） | 较弱、柔和 | ⭐ 较少直接用 |

> 变格终止不是不能用，只是终止感相对柔和。大多数人期待较强的终止感，所以完全终止/正格终止占绝大部分。

### 二、变格补充终止的定位

作曲家根据变格终止「终止感弱、柔和」的特点，挖掘出了它**最典型的用法**：

> **先做完全终止/正格终止（强终止感），再在后方用变格终止做一个小小的补充点缀。**

就像在句号后面加一个柔和的装饰——这就是**变格补充终止**（Plagal Supplementary Cadence）。

```
全终止（S→K₄₆→D→T，强烈的句号）→ 变格补充终止（S→T，柔和的点缀）
```

### 三、两大规则

#### 规则一：必须发生在全终止之后

变格补充终止是「补充」，只有全终止做完了才开始。如果 S→T 出现在全终止之前，那只是音乐进行中的普通一环，不叫补充终止。

```
✅ 正确：... → D → T → S → T
                     ↑全终止  ↑补充
❌ 错误：... → S → T → D → T  （S→T 在完全终止之前，不是补充终止）
```

#### 规则二：最后的 T 必须不弱于前方的 S

| 维度 | 要求 |
|------|------|
| **时值** | 补充终止最后的 T **≥** 前方的 S 或 S₆ |
| **节拍强度** | 补充终止最后的 T **≥** 前方的 S 或 S₆ |

> 「不短于」和「不弱于」包括等于的情况。例如四三拍中 S₆ 和 T 各一拍且都处于弱拍，是允许的。

**为什么？** 因为这个 T 是整个变格补充终止行为中**最重要的和弦**，最需要被强调。如果它短于/弱于前方的 S，就显得仓促和应付。

**反例听觉效果**：
- T 比 S 短 → 草草收场
- T 在弱拍、S 在次强拍 → T 显得不重要，终止感不足
- 两个条件都违反 → 极不自然

### 四、用 S 还是 S₆？

都可以用，没有优先级。试试两种，哪种好听用哪种。

### 五、旋律上的变体

不仅可以通过共同音保持在高声部不动来配变格补充终止，有时旋律本身就直接写成了 S→T 的走向，那自然就只能做变格补充终止。

## 第二部分：低音题技巧升级（第六点）

### 一、核心认知

> **一个低音可以对应多个和弦、多个旋律音。**

低音看似只有一个音符，但可能：
1. 同低音换了和弦（如 K₄₆→D，低音都是 sol）
2. 同和弦内部做旋律变化（如同一 T₆ 内，旋律可以写多个音）

### 二、实操思路

拿到一个低音，思考能不能拆成多个旋律音：

| 低音时值 | 可拆分的节奏 |
|---------|------------|
| 四拍 | 3+1、2+2、1+3、2+1+1 等 |
| 三拍 | 2+1、1+2、1+1+1 等 |
| 两拍 | 1+1、附点节奏（1.5+0.5）等 |
| 一拍 | 二八节奏（0.5+0.5）等 |

### 三、拆分的原则

仍然要遵循第18课的**前五点**（音域、最高音后置、高低反向、平稳为主、跳进后反向），不是乱拆：

- 开头不要一上来就跳太高（最高音留给后半段）
- 拆出的多个旋律音之间尽量平稳
- 跳进后尽量反向
- 平稳进行占多数

### 四、示例对比

- **旧版**（一音对一音）：旋律呆板，节奏单调
- **新版**（运用第六点）：旋律节奏丰富很多，更有呼吸感

## 三、本课总结

### 变格补充终止速查

| 要点 | 内容 |
|------|------|
| 位置 | 全终止之后（D→T 做完后） |
| 形式 | S→T 或 S₆→T |
| 规则 | 最后的 T 时值 ≥ S，节拍 ≥ S |
| 效果 | 柔和的点缀，丰富终止部分 |

### 低音题六要点总览

| 编号 | 要点 | 课次 |
|------|------|------|
| ① | 音域控制（C4~C6） | 第18课 |
| ② | 最高音后置（黄金分割） | 第18课 |
| ③ | 高低反向（约 3:1） | 第18课 |
| ④ | 平稳为主（平稳:跳进 ≈ 3:1） | 第18课 |
| ⑤ | 跳进后反向（约 3:1） | 第18课 |
| ⑥ | 一低音可对应多旋律音 | **第19课（新）** |
