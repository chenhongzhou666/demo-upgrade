## 本课定位

🎯 接上节课 TSⅥ 的第一个用法（属→TSⅥ），展开讲三个新概念：阻碍进行、阻碍终止、扩展乐段。同时详解**属→TSⅥ 的具体连接规则**。

---

## 一、三个新概念

### 阻碍进行

```
定义：属功能和弦 → 非主功能和弦（绝大多数情况 = 属→TSⅥ）
```

D 天然渴望解决到 T。当 D 后面不接 T 时，这个解决过程被「阻碍」了，解决感降低。注意：阻碍≠难听，只是不如 D→T 彻底。

### 阻碍终止

```
定义：阻碍进行发生在「原本应该出现全终止」的位置时
```

在这个位置听众强烈期待 D→T 结束乐段，你却接 TSⅥ → 期待被阻碍 → 音乐必须继续发展。

### 扩展乐段

```
定义：通过阻碍终止（或其他方式）使第二乐句比常规长度更长
```

典型结构：**4(半终止) + X(含阻碍终止+全终止)**，X > 4，形成不等长乐句。

阻碍终止不是扩展的唯一方式——**不完满终止**也可做扩展（但不完满≠阻碍，是两个不同的技术路径）。

---

## 二、属→TSⅥ 连接规则

### 核心铁律

| 规则 | 说明 |
|------|------|
| **TSⅥ 重复三音** | 三音=主音，为了体现 D后TSⅥ 的主功能角色 |
| **导音→主音（优先）** | 属和弦的导音尽量接到 TSⅥ 的主音（三音） |

### 各属和弦接 TSⅥ 速查

| 前方属和弦 | 能否接？ | 关键注意 |
|------|:---:|------|
| **D（属三）** | ✅ | 重复三音，导音→主音 |
| **D₆（属六）** | ✅ 仅大调 | 低音是导音→只能接la（无法满足导音→主音，没关系）；小调禁止（增二度） |
| **D₇ / D₅₆ / D₃₄** | ✅ | 先解决七音（发→咪），再满足重复三音+导音→主音；小调注意增音程 |
| **D₂（属二）** | ❌ 不可 | 七音在低音，只能去咪或保持发，无法去TSⅥ的低音la |

### 小调特别注意

| 连接 | 小调问题 | 考试建议 | 自由创作建议 |
|------|------|------|------|
| D₆→tsⅥ | 低音增二度 | 完全不用 | 可做减七度跳进 |
| D₅₆→tsⅥ | 低音增二度 | 完全不用 | 可做减七度跳进 |
| D₃₄→tsⅥ | 低音可能增音程 | 不用 | 可做减音程跳进 |

## 三、阻碍终止的最佳实践

```
阻碍终止最佳组合：K₄₆ → D₇ → TSⅥ
```

用 K₄₆+D₇ 营造「真的要做全终止了」的氛围，然后突然接 TSⅥ——阻碍感最强。

### 阻碍终止后的推进

TSⅥ 在阻碍终止处体现主功能 → 开启新一轮 T→S→D→T 循环 → 逐步走向真正的全终止。

---

## 四、乐段结构判断更新

有了扩展乐段后，遇到非 8/16 小节的乐段，判断思路：

1. 先尝试**对半划分**（如 10小节的5+5），找依据
2. 找不到→去**第4、第8小节附近**找半终止/阻碍终止依据
3. 找到半终止+阻碍终止→判定为**扩展乐段**（4+N结构）
4. 仍然找不到→考虑单乐句乐段或其他罕见形式

> 不要死记「第4小节=半终止，第8小节=阻碍终止」，小节数只是大概描述，关键是找得到配 D/T 的依据。

---

## 五、总结

| 概念 | 一句话 |
|------|------|
| 阻碍进行 | D→非T（绝大多数=D→TSⅥ），解决感被阻碍 |
| 阻碍终止 | 阻碍进行出现在应该全终止的位置 |
| 扩展乐段 | 通过阻碍终止扩展第二乐句，形成不等长结构 |
| 连接铁律 | TSⅥ 重复三音 + 导音→主音 |
| 最佳组合 | K₄₆→D₇→TSⅥ（阻碍感最强） |
| D₂→TSⅥ | ❌ 不可能（七音在低音，去不了la） |
| 小调慎用 | D₆/D₅₆/D₃₄→TSⅥ 有增音程风险 |
