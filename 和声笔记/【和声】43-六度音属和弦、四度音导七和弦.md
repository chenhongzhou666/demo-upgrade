## 一、六度音属和弦 D⁽⁶⁾

### 来历：拆分 K₄₆ → D

K₄₆ → D 平稳形式的两个解决：

```
K₄₆           D
C (4度音) →  B (3度音)  ← 四度音→三度音的解决
E (6度音) →  D (5度音)  ← 六度音→五度音的解决
G (根音)  =  G (根音)
G (低音)  =  G (低音)

拆分成两步（先做四→三，再做六→五）：

K₄₆ → [D⁽⁶⁾] → D
 C      C       B    ← 第二步才解决四→三
 E      E       D    ← 六→五留到后面
 G      G       G
 G      G       G
```

→ 中间状态 **G-G-B-E** 就是「六度音属和弦」。

### 为什么中间态用 G-G-B-E 而不是 G-G-C-D

| 版本 | 构成音 | 问题 |
|:---:|------|------|
| G-G-B-E | 根-根-三-六 | ✅ 协和、有导音 B 增强属感、E-G-B=小三性质对比 |
| G-G-C-D | 根-根-四-五 | ❌ C-D 二度不协和、缺导音属感弱、无色彩对比 |

> 先做四→三（产生 D⁽⁶⁾），再做六→五；而非先做六→五再做四→三。

### 结构与标记

| 项目 | 内容 |
|------|------|
| **构成音** | 根音 + 根音 + 三音 + **六度音**（重复根音！） |
| **C大调** | G-G-B-E |
| **a小调** | E-E-G♯-C |
| **标记** | D⁽⁶⁾（D 右上方写括号六，区分于 D⁶ 属六和弦） |
| **不是 III 级转位！** | 不理解为 E-G-B 的转位（E-G-B = DTⅢ，与本和弦来历无关） |

### 三个要点

#### ① 六度音几乎总是在高声部

```
✅ E（六度音）在高声部 —— 正常用法（突出六度音的色彩）
⚠️ E 在内声部 —— 极少用，不算错但缺乏意义
```

> 遇到**高声部旋律音 = 调式三级音**（C大调=mi），又想配属功能 → 用 D⁽⁶⁾

#### ② 当成 D₃ 的变体

D₃ 能用的地方、前后能接什么和弦 → D⁽⁶⁾ 同样适用。

| 前后 | 规则 |
|------|------|
| 前面 | K₄₆ / S / SⅡ / T / T₆ / TSⅥ 等（同 D₃） |
| 后面 | T / TSⅥ / D₇ 等（同 D₃），**不可接 S/Ⅱ**（逆功能） |

> 连接方法：优先平稳 → 三步检查（同之前所有和弦）

#### ③ 常见用法汇总

| 场景 | 用法 |
|------|------|
| 全终止 | K₄₆ → D⁽⁶⁾ → D₃ → T |
| 直接使用 | S₆ → D⁽⁶⁾ → T |
| 半终止 | T → D⁽⁶⁾（半终止处） |

---

### 哪些属和弦可以用这个技术

| 和弦 | 可用？ | 原因 |
|:---:|:---:|------|
| **D₃** | ✅ | 原位，五音 re 可换 mi |
| **D₇** | ✅ | 原位，五音 re 可换 mi（但七音仍须解决） |
| **D⁵₆** | ✅ | 三音在低音，五音在上方，OK |
| **D²** | ✅ | 七音在低音，五音在上方，OK |
| **D⁴₆** | ❌ | 仅固定经过搭配，不能擅自改音 |
| **D³₄** | ❌ | 五音 re 在低音，换上 mi = 六度音在低音（矛盾） |
| **D⁹** | ❌ | 大多数省略五音，且同时突出九音和六度音=主次不明 |
| **D⁶** | ❌ | 无七音→属感弱，再换 mi 更模糊 |

### D₇⁽⁶⁾ 的特殊注意

有七音 → **七音发必须 ↓ 级进解决到 mi**（不可变）。因此 D₇⁽⁶⁾ → T 的连接选择有限：

```
G-B-F-E → C-E-G-C （七音 F→E，六度音 E→D→在下一和弦是C）
```

> 阻碍终止处：D₇⁽⁶⁾ → TSⅥ，TSⅥ 必须重复三音。

---

### C大调属和弦覆盖所有旋律音

学了 D⁽⁶⁾ 之后，**除了 do 以外的所有音都可以配属功能和弦了**：

| 旋律音 | 属功能和弦 |
|:---:|------|
| sol | D₃, D₇, D₉ |
| si | D₃, D₇, D⁵₆ |
| re | D₃, D⁷, D² |
| fa | D₇, D³₄, D² |
| la | D₉ |
| **mi** ⭐ | **D⁽⁶⁾, D₇⁽⁶⁾** |
| do | ❌（主音，不配属功能——除非 K₄₆） |

> 全终止处的旋律选择极大丰富：sol-do / re-do / si-do / fa-mi / la-sol / **mi-re** 等。

## 二、四度音导七和弦 DVII⁽⁴⁾

### 完全来源于六度音属和弦

同样的 re→mi 技术「传染」到导七：

```
C大调 DVII₇:    B-D-F-A   →   B-E-F-A   （D换成E）

E 对属音 G 是六度 → D⁽⁶⁾
E 对导音 B 是四度 → DVII⁽⁴⁾（标记：DVII 右上方写括号四）
```

### 要点

| 项目 | 内容 |
|------|------|
| 使用频率 | 比 D⁽⁶⁾ 更罕见 |
| 前后接法 | 完全参考正常 DVII₇ |
| 七音 | 仍需 ↓ 级进解决或横向保持 |
| E 的位置 | 几乎总在高声部 |
| 可用形式 | DVII₇⁽⁴⁾, DVII³⁴⁽⁴⁾, DVII₂⁽⁴⁾ |
| 不可用形式 | DVII₆, DVII⁵₆（五音 re 在低音，冲突） |
