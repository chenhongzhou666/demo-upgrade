## 细节一：次强拍与弱拍也可更换和弦

### 之前

每小节只配 1 个和弦，仅在强拍更换。

### 升级后

| 拍位 | 可换和弦？ | 频率 |
|------|:---:|:---:|
| **强拍** | ✅ | 几乎都换 |
| **次强拍** | ✅ | 中等（比强拍少） |
| **弱拍** | ✅ | 最低（偶尔） |

### 什么时候换

不看节拍位置，**看旋律和音乐结构**：

```
例：第二小节旋律 = fa la fa | mi sol
    → 前半 fa-la-fa 配 S，后半 mi-sol 配 T
    → 在次强拍（第三拍）换和弦
```

```
例：倒数第二小节旋律 = do si（全终止处）
    → do 配 K₄₆，si 配 D₇
    → 即使在弱拍，该换也得换
```

### 核心原则

> **更换和弦频率：强拍 > 次强拍 > 弱拍**

原因 = 音乐信息量概念（第12课）：强拍信息量多（换新和弦=新信息），弱拍信息量少。反着来 → 强弱律动崩塌。

## 细节二：延音踏板的「同和弦中途换踏板」

### 之前

换和弦 = 换踏板。

### 升级后

同一个和弦持续太久 → 听感浑浊 → 可以中途松一下再踩。

### 关键规则

> **在最低音重新弹响的地方换踏板。**

```
✅ 柱式：第一拍弹低音，第三拍又弹低音 → 在第三拍换（低音刚好重启）
✅ 分解：低音只在第一拍弹了一次 → 不建议中途换（换了低音消失）

❌ 随便找一拍换 → 低音突然缺失，前后饱满度断层
```

### 为什么

- 在低音重新弹响时换 → 换踏板前有低音延续（饱满），换踏板后立即有新的低音（继续饱满）
- 在低音没有重弹时换 → 踏板一松，低音消失，后方只有高处稀疏的音

## 细节三：模式一半终止 K₄₆→D 不再强制「双解决」

### 模式零要求（半终止处）

K₄₆→D 必须**同时做**四度→三度 + 六度→五度（才有逗号感）。

→ 只有旋律是 **mi-re / do-si / sol-sol** 三种情况才能配 K₄₆→D。

### 模式一放宽

**只要和弦能配成 K₄₆→D 就可以用**，即使没同时做双解决。

```
例：半终止处旋律 = mi si
    → 模式零：不能配 K₄₆→D（只能配 T→D）
    → 模式一：可以配 K₄₆→D（仍有断句感，只是稍弱）
```

> 本质：模式零的严格规则是为了让你**理解**双解决的听感，理解后在实际创作中就可以放宽。

## 细节四：结尾丰富化 ⭐

### 4-1：最后一个和弦不必与旋律音同高

```
之前：旋律最后音 = do → 右手最后一排最高音也是 do
升级：右手最后和弦最高音可以是 mi / sol / do（主和弦任意构成音）
```

| 结尾高度 | 听感 |
|:---:|------|
| 高 | 轻盈、飘逸 |
| 低 | 沉淀、稳重 |
| 琶音 | 灵动、温柔 |

> 比「右手最后一排和旋律音一模一样」更丰富，不再单调。琶音（波浪线符号）见乐理第二季第92课。

### 4-2：两结尾 T 之间用「非 T 音」衔接

```
倒数第二小节（旋律结束）→ 衔接段 → 最后一个柱式T（伴奏结束）
   红框①                  蓝框         红框②
   必须用T音            可用re/fa/la/si  必须用T音
   （配合旋律终止）     （自由发挥！）   （伴奏终止）
```

**衔接段（蓝框）可以做的事：**

| 用哪些音 | re / fa / la / si 里选 1-3 个（四个全用较难做好听） |
|------|------|
| 用什么节奏 | 可变化（不限于二八），这本来就是「适合变化」的位置（第37课） |
| 效果 | 对比感 + 新鲜感，比全用 do-mi-sol 有趣 |

**例子：流行歌最爱用的收尾方式**

```
倒数第二小节：右手弹 D-F 的衔接（蓝框）
最后一个 T：  右手最高音 = E（三音旋律位置）

→ re-fa 对比 do-mi-sol 新鲜
→ mi 旋律位置 = 不完满终止 = 开放式结局感
→ 流行情歌最爱（不坚定结束，留余韵）
```

### 衔接段的标记问题

| 看法 | 做法 | 理由 |
|------|------|------|
| 精确派 | 标 T₉ / Cmaj9 等 | 把所有音算进和弦 |
| 重点派 | 标 T / C | 红框（重点）才是 T，蓝框只是衔接音 |

> 两种看法都没错，**不要纠结标记**，真正重要的是理解技法。本课统一标 T/C。

## 总结：四个优化一览

| # | 优化点 | 一句话 |
|:---:|------|------|
| ① | 次强拍/弱拍换和弦 | 看旋律需求，但频率：强>次强>弱 |
| ② | 同和弦中途换踏板 | 在低音重弹时换，不能随便换 |
| ③ | 半终止 K₄₆→D 放宽 | 不必强制双解决，能配就配 |
| ④ | 结尾丰富化 | 最后和弦可高低变化 + 衔接段用非 T 音 + 琶音 |

## 练习要求

- **模式零**：高音题 + 低音题各一道（巩固上节课）
- **模式一**：一道旋律，柱式/分解/混合各配一个
  - 必须思考四个优化点，特别是**结尾丰富化**需要多尝试
