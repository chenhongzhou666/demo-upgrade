## 本课定位

🎯 学习**经过的四六和弦**——三和弦第二转位（四六和弦）用于经过性质的和声连接。本课涉及两个组合：属四六（D₄₆）和主四六（T₄₆），六条规则完全对称。

---

## 一、基本概念

### 四六和弦 = 三和弦第二转位 = 五音在低音

| 示例（C大调） | 原位 | 第一转位（六和弦） | 第二转位（四六和弦） |
|:---:|:---:|:---:|:---:|
| T | C-E-G（低音C） | E-G-C（低音E）→ T₆ | G-C-E（低音G）→ T₄₆ |
| D | G-B-D（低音G） | B-D-G（低音B）→ D₆ | D-G-B（低音D）→ D₄₆ |

### 「经过」意味着什么

使用「经过」这个词，说明这种和弦具有经过音的两种特质：

| 特质 | 含义 |
|------|------|
| **不强于两侧** | 所处位置节拍强度/时值不高于两侧和弦（弱拍弱位） |
| **低音级进** | 低音与左右两侧和弦的低音是连续的级进关系 |

---

## 二、组合一：T → D₄₆ → T₆（及反向 T₆ → D₄₆ → T）

### 构造（C大调示例）

```
T:    低音 do  上方: mi-so-do（根音+三音+五音各一）
D₄₆:  低音 re  上方: fa-so-ti（根音+三音+五音各一）→ 重复五音！
T₆:   低音 mi  上方: so-do-mi（根音+五音+三音各一）
```

### 声部进行（最平稳形式）

| 声部 | T → D₄₆ → T₆ | 进行距离 |
|------|:---:|:---:|
| 高声部 | mi → re → do | 全是二度级进 |
| 中声部 | so → so → so | 共同音保持 |
| 次中声部 | do → ti → do | 小二度+小二度 |
| 低声部 | do → re → mi | 全是二度级进 |

> 四个声部全部不超过二度！这就是经过四六和弦的核心魅力：**极致的平稳过渡**。

### 为什么要这样构造？

**为什么重复五音（D₄₆ 低声部五音 + 上方声部各一个根三五音）？**

因为只有这样才能让四个声部同时做到最平稳的连接。如果重复根音或三音，必然有某个声部会产生大于二度的进行。

### 六条规则

| 规则 | 内容 |
|------|------|
| **①** 不强于两侧 | D₄₆ 必须在弱拍/弱位（经过性质） |
| **②** 低音级进 | 三个和弦的低音为连续级进（do-re-mi 或 mi-re-do） |
| **③** 重复五音 | D₄₆ 必须是五音在低音 + 上方根三五音各一 |
| **④** 正反皆可 | T→D₄₆→T₆ 和 T₆→D₄₆→T 都可以用 |
| **⑤** 上方轮转 | 上方三声部（mi-re-do、so-so-so、do-ti-do）谁在高声部都行 |
| **⑥** 六和弦可变 | T₆ 有时可以重复五音（do-ti-do → sol-ti-do），视前后联系而定 |

### 如何识别该用？

当旋律出现以下**级进三音组**时，想到可以用这套组合：

| 旋律形态 | 对应和声 |
|---------|---------|
| 1→2→3 级（do-re-mi） | T₆→D₄₆→T |
| 3→2→1 级（mi-re-do） | T→D₄₆→T₆ |
| 1→7→1 级（do-ti-do） | T→D₄₆→T₆ |
| 连续五级音（so-so-so） | 可在下方配 T→D₄₆→T₆（共同音保持） |

> 数字表示**级数**，不是固定调唱名。如 E大调的 1-2-3 = E-F#-G#。

## 三、组合二：S → T₄₆ → S₆（及反向 S₆ → T₄₆ → S）

### 构造（C大调示例）

```
S:    低音 fa  上方: la-do-fa
T₄₆:  低音 sol 上方: do-mi-sol → 重复五音！
S₆:   低音 la  上方: do-fa-la
```

### 声部进行（最平稳形式）

| 声部 | S → T₄₆ → S₆ | 进行 |
|------|:---:|:---:|
| 高声部 | la → sol → fa | 全级进 |
| 中声部 | do → do → do | 共同音保持 |
| 次中声部 | fa → mi → fa | 小二度往返 |
| 低声部 | fa → sol → la | 全级进 |

### 所有规则与组合一完全一样

只是 D₄₆ 换成 T₄₆，T/T₆ 换成 S/S₆。

| 旋律形态 | 对应和声 |
|---------|---------|
| 6→5→4 级（la-sol-fa） | S→T₄₆→S₆ |
| 4→5→6 级（fa-sol-la） | S₆→T₄₆→S |
| 4→3→4 级（fa-mi-fa） | S→T₄₆→S₆（同 do-ti-do 是声部之一轮转来的） |
| 连续一级音（do-do-do） | 可在下方配 S→T₄₆→S₆ |

## 四、经过和弦速查表

| 组合 | 低音 | 上方三声部（可轮转） | 六和弦可变体 |
|------|:---:|------|------|
| T→D₄₆→T₆ | 1→2→3 | mi-re-do / do-do-do / do-ti-do | do-ti-do → sol-ti-do |
| T₆→D₄₆→T | 3→2→1 | do-re-mi / do-do-do / do-ti-do | 同上 |
| S→T₄₆→S₆ | 4→5→6 | la-sol-fa / do-do-do / fa-mi-fa | fa-mi-fa → do-mi-fa |
| S₆→T₄₆→S | 6→5→4 | fa-sol-la / do-do-do / fa-mi-fa | 同上 |

> ⚠️ 表中数字均为**级数**（首调），不是固定调唱名。上方三声部可任意轮转，谁都可以做高声部旋律。

---

## 五、关键辨析：T₄₆ ≠ K₄₆（同一构成音，完全不同功能）

在 C 大调中，T₄₆ 和 K₄₆ 的构成音完全一样（低音 G + 上方 C-E-G），但**功能完全不同**：

| | T₄₆（经过主四六） | K₄₆（终止四六） |
|------|------|------|
| **前后环境** | S ↔ S₆（下属功能环境） | 半终止/全终止处，接原位 D |
| **节拍位置** | 弱拍/弱位（经过过渡） | 强拍/次强拍（制造紧张） |
| **听感** | 下属功能内的平稳过渡 | 属功能的不稳定、急需解决 |
| **重复音** | 重复五音 | 重复根音（属音） |

> 核心意识：**同一个和弦的构成音，因所处位置、前后和弦、强弱不同，可能具有完全不同的功能和听感。** 这和等音程的道理类似——单独听是一样的，但在音乐上下文中的感受不同。

## 六、实战流程更新

拿到一条旋律，现在要做的事更多了：

1. 判断调性
2. 标出半终止/全终止/K₄₆ 等特殊结构
3. 看能否做变格补充终止（第19课）
4. **看有哪些地方可以做经过四六和弦**（第20课新）——对照速查表
5. 从头开始配和声

> 随着学的技巧越来越多，看到旋律时能想到的和声可能性也越来越多。这就是「旋律与和声不分家」。

---

## 七、本课总结

| 要点 | 一句话 |
|------|--------|
| D₄₆ 经过 | T ↔ T₆ 之间插入，低音 do-re-mi（或反向），极致平稳 |
| T₄₆ 经过 | S ↔ S₆ 之间插入，低音 fa-sol-la（或反向），同理 |
| 共同特征 | 弱位、重复五音、低音级进、上方轮转、正反皆可 |
| T₄₆ vs K₄₆ | 构成音相同，但位置/环境不同导致完全不同的功能 |
