## 为什么需要功能体系？

**和弦固定标记法（C、Cm、G7...）的局限**：只告诉你弹哪些音，不告诉你这个和弦在调里扮演什么角色。

同一个 Cm 和弦：
- 在 **c 小调** 中 → 主和弦（T），稳定、结束感
- 在 **G 大调** 中 → iv 级，对比、新奇感，绝不是结束

> **核心思想**：「你具体是什么和弦不重要，在调里是什么角色才重要。」

功能体系用同一种标记理解同一角色的和弦，方便分析不同调的和声逻辑。

## TSD 三大功能

```
         T（主功能）
        /          \
   S（下属）  ←→  D（属）
```

### T — 主功能（Tonic）

| 属性 | 说明 |
|------|------|
| 级数 | I 级（一级主音为根音构建的调内三和弦） |
| 标记 | 大写 **T**（大调）、小写 **t**（小调） |
| 感受 | **稳定、在家、音乐结束了** |
| 位置 | 绝大多数音乐从 T 出发，最终回到 T |

例：
- C 大调的 T = **C E G**（大三和弦）
- 降A大调的 T = **降A C 降E**（大三和弦）
- a 小调的 t = **A C E**（小三和弦）

> 不同调的具体和弦不同，但都是 I 级，感觉一样——这就是功能体系的核心。

### D — 属功能（Dominant）

| 属性 | 说明 |
|------|------|
| 级数 | V 级（五级音为根音） |
| 标记 | 大写 **D**（大调本就大三，小调升7级后也大三） |
| 感受 | **紧张、迫切想回家、推动音乐进行** |
| 词源 | Dominant = 占支配地位的、主导的 |

**D → T 称为「解决」**（从紧张→稳定）

例：
- C 大调的 D = **G B D**
- E 大调的 D = **B #D #F**

#### 小调中的特殊处理

自然小调的五级本是小三和弦（如 a 小调：E G B），缺少「导音→主音」的小二度紧张感，所以 **实际运用中几乎总是升高 VII 级音（五级和弦的三音）**，变成大三和弦 → 标记大写 D。

> 这就是「和声小调」比「自然小调」更常用的原因。

```
自然小调 d→t：E G B → A C E  （柔美，缺紧迫感）
和声小调 D→t：E #G B → A C E （#G→A 小二度，急迫回家）
```

### S — 下属功能（Subdominant）

| 属性 | 说明 |
|------|------|
| 级数 | IV 级（四级音为根音） |
| 标记 | 大写 **S**（大调）、小写 **s**（小调） |
| 感受 | **刚出门、刚起劲、还不够尽兴** |
| 词源 | Sub（下方）+ dominant（属）= 下属 |

例：
- C 大调的 S = **F A C**
- 降B大调的 S = **降E G 降B**

## 三种循环的感受对比

| 循环 | 比喻 | 内部走向 |
|------|------|---------|
| **T → D → T → D** | 在家→出去尽兴了→急迫冲回家 | 音往上走（主动感） |
| **T → S → T → S** | 在家→刚出门→被叫回家 | 音往下走（被迫感） |
| **T → S → D → T** | 完整旅程：出发→玩→尽兴→回家 | 经典功能进行 |

## 课后练习

任选几个大小调，写出它们的 T、S、D 分别是哪三个和弦，用**音名（固定调）**而非首调唱名。

大调公式：**I 级、IV 级、V 级音分别构建大三和弦**
小调公式：**I 级、IV 级音构建小三和弦，V 级构建大三和弦**（升 VII 级）

> 无需死记，推导多了自然记住。

### 一、所有大调的 T / S / D

> 规则：I、IV、V 级均构建**大三和弦**

**升号调（按五度圈升序）：**

| 调性 | 调号 | T（I 级） | S（IV 级） | D（V 级） |
|------|------|-----------|------------|----------|
| C 大调 | 0# | C—E—G | F—A—C | G—B—D |
| G 大调 | 1# (F#) | G—B—D | C—E—G | D—F#—A |
| D 大调 | 2# (F# C#) | D—F#—A | G—B—D | A—C#—E |
| A 大调 | 3# (F# C# G#) | A—C#—E | D—F#—A | E—G#—B |
| E 大调 | 4# (F# C# G# D#) | E—G#—B | A—C#—E | B—D#—F# |
| B 大调 | 5# (F# C# G# D# A#) | B—D#—F# | E—G#—B | F#—A#—C# |
| F# 大调 | 6# (F# C# G# D# A# E#) | F#—A#—C# | B—D#—F# | C#—E#—G# |
| C# 大调 | 7# (全部升) | C#—E#—G# | F#—A#—C# | G#—B#—D# |

**降号调（按五度圈降序）：**

| 调性 | 调号 | T（I 级） | S（IV 级） | D（V 级） |
|------|------|-----------|------------|----------|
| F 大调 | 1b (Bb) | F—A—C | Bb—D—F | C—E—G |
| Bb 大调 | 2b (Bb Eb) | Bb—D—F | Eb—G—Bb | F—A—C |
| Eb 大调 | 3b (Bb Eb Ab) | Eb—G—Bb | Ab—C—Eb | Bb—D—F |
| Ab 大调 | 4b (Bb Eb Ab Db) | Ab—C—Eb | Db—F—Ab | Eb—G—Bb |
| Db 大调 | 5b (Bb Eb Ab Db Gb) | Db—F—Ab | Gb—Bb—Db | Ab—C—Eb |
| Gb 大调 | 6b (Bb Eb Ab Db Gb Cb) | Gb—Bb—Db | Cb—Eb—Gb | Db—F—Ab |
| Cb 大调 | 7b (全部降) | Cb—Eb—Gb | Fb—Ab—Cb | Gb—Bb—Db |

**等音调（键盘上同音，记谱不同）：**
- B 大调 (5#) = Cb 大调 (7b)
- F# 大调 (6#) = Gb 大调 (6b)
- C# 大调 (7#) = Db 大调 (5b)

### 使用说明

1. **做课后练习时**：自选大小调写出 TSD，然后用上表核对
2. **不需要死记**：推导多了自然记住（I/IV/V 级构建规则 + 调号决定升降）
3. **小调 V 级特别注意**：永远是**大三和弦**（升 VII 级），所以标记为大写 D
4. **日常使用**：最常接触的是 0-4 个升降号以内的调，边缘调（5-7 个升降号）较少出现
