## 补充：异质化音色不怕被高音掩盖

### 问题场景

当主旋律由低音乐器/声部演奏（大提琴、男低音、大号等），旋律主要在**大字组~小字组**——如果钢琴右手照抄同样高度，左手只能更低 → 大量下加线 + 严重浑浊。

```
❌ 大提琴奏 C3-E3，钢琴右手也弹 C3-E3 → 钢琴太低 → 浑浊、糊
✅ 钢琴右手提高八度，弹 C4-E4 → 钢琴在舒适区 → 清晰
```

### 为什么不会抢戏

| 前提 | 规则 | 适用？ |
|------|------|:---:|
| **同质化音色**（全是钢琴/全是弦乐） | 人耳注意到最高的声音 ✓ | ✅ 适用 |
| **异质化音色**（大提琴 + 钢琴） | 人耳能轻易分辨不同音色 | ❌ 不适用 |

> 只要不刻意把主旋律乐器音量压得很小，钢琴弹得比它高也不会抢戏。异质化音色的区分度远大于音高差异。

### 钢琴的舒适音区

| 手 | 推荐主要范围 |
|------|------|
| 右手 | 小字一组 ~ 小字三组（C4 ~ C7） |
| 左手 | 大字组 ~ 小字一组（C2 ~ C4） |

> 即使偶尔超出也没关系，大部分音落在这两个区域即可。

## 第31-38课 结构化梳理

### 阶段一：基础框架（第31课）

**第31课 — 模式一入门：和弦选择**

| 要点 | 内容 |
|------|------|
| 模式零 vs 模式一 | 一个旋律音配一个和弦 → 一个和弦可对应多个旋律音 |
| 和弦选择三要素 | ① 位置（首尾T/半终止D/全终止D→T）<br>② 音符占比 ≥ 50%<br>③ 强拍占比 ≥ 50% |
| 本课限制 | 每小节1和弦、仅原位、三度叠置狭义形态 |

---

### 阶段二：方法论与思维（第32-33课）

**第32课 — 控制变量思维**

| 要点 | 内容 |
|------|------|
| 和弦选择三大场景 | A 旋律音=和弦一部分(看位置+164251)<br>B 刚好构成和弦<br>C 无法构成(用三要素+功能逻辑) |
| 功能逻辑升级 | **164251**，优先保证 T-S-D 三种功能都在 |
| 核心方法论 | **一次只改变一个因素**（节奏/织体/纵向排列等） |
| 节奏5层级 | 全音符 → 覆盖每拍 → 每拍一分为二 |
| 织体 | 柱式(氛围感) vs 分解(低→高循环，流动清晰) |

**第33课 — 思考依据做出选择**

| 要点 | 内容 |
|------|------|
| 三步工作流 | ① 选择和弦 → ② 控制变量（多版本） → ③ **思考依据缩小范围** |
| 节奏四依据 | ① 复拍子覆盖次强拍 ② 结尾在强拍/次强拍 ③ 次强拍优先于非强拍 ④ 结尾多用疏节奏 |
| 核心循环 | 对比听 → 判断好坏 → 猜测原因 → 验证 → 提炼依据 |

### 阶段三：新元素引入（第34-35课）

**第34课 — 新因素与新补充**

| 要点 | 内容 |
|------|------|
| 结尾刹车感两招 | ① 提前降密度 ② 伴奏多写1-2小节收尾 |
| 延音踏板（右） | 分解多用（防单薄）、柱式少用（防浑浊）；**换和弦必换踏板** |
| 弱起小节 | 柱式仅弹构成音（不重复低音），旋律进入后再完整织体 |
| 右手旋律+伴奏 | 旋律在上方高音区保持清晰；纵向排列七大要点 |

**第35课 — 分解织体优化**

| 要点 | 内容 |
|------|------|
| 核心问题 | 上密下疏 + 低→高分解 → 左手逼近右手 |
| 解法一 | 降低起点（低1-2八度）→ 厚重/浑浊 |
| 解法二 | 原路折返 → 轻盈/干净 |
| 拍号区别 | 单拍子（四二/四三/八三）低音仅强拍一次；复拍子（四四/八六）次强拍可重启 |

---

### 阶段四：织体总表（第36课）

**第36课 — 织体的具体形式**

| 类型 | 种类数 | 核心 |
|------|:---:|------|
| 柱式 | 5种 | ①每拍相同 ②低音+重复高音 ③低→高 ④高→低 ⑤强位重现 |
| 分解 | 5种 | ①一直往上 ②往上后折返 ③方向自由 ④阿尔贝蒂低音 ⑤Z字形 |
| 混合 | 3种 | ①先分解后柱式 ②先柱式后分解 ③交叉混合 |

**总原则**：低音优先、复拍子可选次强拍重启、低音可叠八度、**分解节奏 ≥ 柱式节奏**

---

### 阶段五：审美核心（第37课）

**第37课 — 统一与变化的运用 ⭐**

| 维度 | 规则 |
|------|------|
| **比例** | 统一:变化 = N:1 ~ 3:1（N>3），禁忌 ≤ 2:1 |
| **位置** | 变化优先放乐句后半段：全终止 > 半终止 > 终止前一小节 |
| **四种因素** | ① 织体类型 ② 同类型具体写法 ③ 节奏疏密 ④ 和弦数量（K₄₆→D 引入） |
| **平行乐段规律** | 开头相似度80-100%→末尾0-20%，越相似越要统一 |

---

### 阶段六：细节提升（第38课）

**第38课 — 模式零规则降级应用于模式一**

| 模式零规则 | 模式一对应 |
|------|------|
| 声部交叉 | 左手别超右手 |
| 声部超越 | 分微观+宏观，分层检查 |
| 五八度 | ⏭️ 暂跳过（等转位） |
| 重复音错误 | 根五为主，避三音七音 |
| 上方超纯八度 | 上密下疏 |
| 四部同向 | 别所有层次同方向走 |
| 增音程/对斜 | ⏭️ 暂跳过（等转位+和声大调） |
| 七音未解决 | SⅡ₇/D₇ 的七音→下一和弦同组回应 |

> 核心：从「绝对不能犯」降级为「尽量不犯」（有理由时可以犯）

---

## 异质化音色补充总结

| 场景 | 钢琴右手做法 | 原因 |
|------|------|------|
| 主旋律在高音区（女高音/小提琴） | 照抄同样高度 | 钢琴本身就在舒适区 |
| 主旋律在中低音区（男声/大提琴） | **提高八度**到小字一组以上 | 异质化音色不抢戏 + 避免钢琴浑浊 |
| 主旋律在极低音区（男低音/大号） | 可提高两个八度 | 同上 |
