## 本课定位

🎯 打通模式零和模式一。模式零的四部和声规则不是教条——把它们从「绝对不能犯」降级为「尽量不犯」，就是模式一从 90 分到 95 分的细节提升之道。

---

## 一、模式零 → 模式一：规则的降级

| 模式零 | 模式一 |
|------|------|
| 括号内错误**绝对不能犯** | 括号内错误**尽量不犯**（有理由时可以犯） |
| 优先平稳进行 | 优先平稳进行 |
| 优先外声部跳进 | 优先外声部跳进 |
| 优先更少声部跳进 | 优先更少声部跳进 |

> 为什么降级？模式一不是考试，偶尔犯且觉得更好听 → 完全可以犯。但当没有特殊理由时，遵守规则会让细节更好。

---

## 二、八大规则的逐一对照

### ① 声部交叉

**模式零**：任何两声部交叉 = 错误。

**模式一**：几乎不会发生（不分 SATB 声部标注）。唯一等价的注意点——**左手不要弹得比右手高**（之前已掌握）。

---

### ② 声部超越

**模式零**：上方声部低于下方声部先前音 = 错误。

**模式一**：分微观和宏观两种情况。

#### 微观（同一和弦内部）

以柱式织体为例——同一小节后几拍如果在不同音高位置弹同一和弦，要注意：

```
❌ 第2拍最高音=mi，第3拍最高音=sol → 声部超越（mi被sol超越）
   → 听众会听出「米索米」而非你想要的「米多米」

✅ 保持各层次的高度稳定，让听众明确听到你想要的高音旋律线
```

**核心认知**：声部超越导致听众对音高线条的判断模糊。是否接受取决于你的需求。

**取舍示例**（同一段四种写法）：

| 版本 | 最高音线 | 声部超越 | 三音重复 | 厚度 | 选它的理由 |
|:---:|:---:|:---:|:---:|:---:|------|
| A | 米多米 ✅ | ❌ 有 | ❌ | 厚 | 要最高音线+接受模糊 |
| B | 米索米 | ✅ 无 | ❌ | 厚 | 无所谓最高音线 |
| C | 米多米 ✅ | ✅ 无 | ❌ 有 | 厚 | 要最高音线+接受三音重复 |
| D | 米多米 ✅ | ✅ 无 | ✅ 无 | 薄 | 都不能接受→牺牲厚度 |

> 没有完美答案。重要的是**你思考过每种写法的理由**，然后根据自己的需求取舍。

#### 宏观（多个和弦之间）

**柱式织体**：观察不同和弦的同一「层次」。如每小节第一拍=低音层，后三拍=柱式和弦层。柱式和弦层之间不应超越。

```
❌ 第1小节和弦层高，第2小节突然变低 → 不自然（听觉暂留导致）
✅ 每个层次各自平稳，不要忽高忽低
```

**分解织体**：纵向出现几个音就分几层（用颜色线条标注），然后检查相邻小节间是否有超越。

**层次数不等时怎么检查**？从外向里：
- 最高层不比前一节第二高层更低 ✓
- 第二高层不比前一节第三高层更低、不比前一节最高层更高 ✓
- 对称地从最低层同样检查 ✓

### ④ 重复音错误

**模式零**：三和弦重复根音，六和弦不重复三音等（严格）。

**模式一**：已经掌握——纵向 2-6 声部合理，**重复根音/五音为主，尽量不重复三音/七音**，强拍次强拍多音头、弱拍少音头。不再赘述。

---

### ⑤ 上方三声部相邻超过纯八度

**模式零**：严格错误。

**模式一**：尽量满足**上密下疏**。不要求时刻满足，但大部分地方应该做到。

**柱式织体**：看到「左手弹了 3-4 个音、右手只有一个音」→ 检查右手是否与左手最高音相距超过纯八度。

```
❌ 右手单音 + 左手密集排列 → 纵向失衡，浑浊
✅ 右手加音（上密）+ 左手减音（下疏）→ 清晰自然
```

**分解织体**：分层观察后，保持上方层次间距小、下方层次间距大。

```
❌ 上疏下密 → 头重脚轻
✅ 上密下疏 → 根基稳固
```

---

### ⑥ 四部同向

**模式零**：四个声部同时上行或下行 = 错误。

**模式一**（抽象为「所有声部/所有层次朝同一方向运动」）：

```
❌ 第2小节所有层次都比第1小节低 → 伴奏突然变浑浊
✅ 每次换和弦时至少有一个层次不动或反向走
```

**原因**：伴奏是配角，应形成**稳定的基调**。所有层次一起往下=突然变浑浊，所有一起往上=突然变单薄——除非你有特殊目的，否则无意义。

**柱式织体**：观察到四部同向→调整纵向排列（改变音高位置）来打破。

**分解织体**：先分层，再看是否所有层次都朝一个方向走。

### ⑧ 七音未解决或保持

**模式零**：严格错误。

**模式一**：用了七和弦（目前仅 SⅡ₇ / D₇）→ **尽量解决或保持其七音**。

**规则回顾**：
- 接属功能 → 七音 ↓ 级进解决
- 接主功能 → 横向保持

**模式一的具体做法**：不要求「每一个出现的七音都要解决」，而是——

1. 确定七音在**哪个音组**（如小字一组 C、小字组 F 等）
2. 下一和弦在**对应音组**至少出现一次解决音/保持音
3. 因为「听觉暂留」，七音即使只出现过一次，脑海里也留下了需要解决的印象
4. 哪个声部弹解决音不限（右手/左手都可以），只要弹了那个琴键就行

```
❌ D₇ 七音(fa)在小字组，后面的T在小字组没弹mi → fa悬空悬在那里
✅ T 至少出现一个小字组的mi → fa→mi 解决感完整
```

> 差别非常细微，初学听不出很正常。先记住原理，耳朵灵敏后会感受到。

## 三、实战流程总结

**做模式一的练习时**，在完成和弦选择+织体+统一与变化后，再过一遍这些细节：

```
① 左手别超右手（声部交叉）
② 分层检查声部超越（微观+宏观）
③ 重复音：根五为主，避三音七音
④ 上密下疏（右手别太孤高）
⑤ 别所有层次一起往一个方向走（四部同向）
⑥ 七和弦的七音要回应（解决/保持）
⑦ 暂不检查：五八度、增音程、对斜（等转位引入后再说）
```

**核心心态**：不是要你全部做到（不现实），而是**每个地方都思考过**。有理由 → 可以犯。没理由 → 尽量做。

## 四、练习要求

- **模式零**：高音题 + 低音题各一道
- **模式一**：一道旋律，柱式/分解/混合各配一个
  - 照常做和弦选择 + 织体 + 统一与变化
  - **新增**：按本课讲的方法逐一过细节（声部超越、上密下疏、四部同向、七音解决）
  - 思考每个取舍的理由
