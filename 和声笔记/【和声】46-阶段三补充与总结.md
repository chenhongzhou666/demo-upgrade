## 一、第25-30课 易错补充

### 易错①：原位 SⅡ₃ 仅大调用

| 调式 | 原位 SⅡ₃ | 可用？ | 原因 |
|------|------|:---:|------|
| 大调 C | D-F-A（小三和弦）| ✅ | 协和 |
| 小调 a | B-D-F（减三和弦）| ❌ | 减三不用原位 |
| 大调用和声大调 | D-F-A♭（减三）| ❌ | 降VI后变成减三，也不可用 |

### 易错②：I 级 ↔ II 级的平行五度

C大调中 T/T₆/T₄₆ 都有 C-G，SⅡ系列都有 D-A → C→D + G→A = **平行纯五度**

**两种解法：**

| 解法 | 做法 |
|:---:|------|
| ① 换声部位置 | 让「G→A」在「C→D」下方 → 变成平行四度（可以） |
| ② 和声大调 | II 里用 A♭ → 变成纯五度↔减五度（课程里不算错） |

### 易错③：TSⅥ 不能用和声大调降VI

TSⅥ（C大调 A-C-E）降VI → A♭-C-E = 增三和弦。根音变了+增三不用 → 禁止。

### 易错④：阻碍终止处 TSⅥ 必须重复三音

原因见第30课。忘了回去看。

## 二、第40-43课 易错补充

### 易错⑤：原位导三无论大小调都不用

C大调 B-D-F 和 a小调 G♯-B-D 都是减三和弦 → 不用原位。最多用 DVII₆。

### 易错⑥：小调不用 DTⅢ

a和声小调 DTⅢ = C-E-G♯（增三和弦）→ 不用。等学到自然小调再说。

### 补充①：半终止 K₄₆ 后面不能接 D⁽⁶⁾

第43课漏讲了。半终止处若要 K₄₆，后面**只能接 D₃**（或少数情况 D₂）——必须同时做四→三+六→五解决，才有句号感。D⁽⁶⁾ 不能替代。

> 半终止不用 K₄₆ 的话，可以单独用 D⁽⁶⁾。

### 补充②：同功能组和弦相接，七音尽量解决/保持

```
DVII⁵₆ → D³₄ : 七音尽量解决/保持（做了更好，不做也不算错）
SⅡ⁵₆ → S   : 同上
```

> 不强制，但做了细节更好。

## 三、第31-39课 + 44-45课（模式一）总结

模式一共11课，核心记住一个概念：

> **统一与变化。** 无论织体/节奏/音高/加花，大部分地方统一、少部分地方变化（半终止/全终止处）。其余规则都是模式零的灵活降级版。
