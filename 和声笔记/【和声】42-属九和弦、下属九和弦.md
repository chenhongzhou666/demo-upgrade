## 本课定位

🎯 在属七和 SⅡ₇ 基础上再叠一个三度 = 九和弦。本课学两个：属九（D₉）和下属九（SⅡ₉）。核心规则就一条——**根音与九音必须相距 ≥ 九度**。

---

## 一、属九和弦（D₉）

### 1. 结构与标记

**C大调属九：G-B-D-F-A**（在属七上再叠三度）

| 调式 | 结构 | 九度类型 | 标记 |
|------|------|:---:|:---:|
| 自然大调 | G-B-D-F-A | 大九度 | D₉ |
| 和声大调（降VI） | G-B-D-F-A♭ | 小九度 | D₉（或 D₉°） |
| 和声小调 | E-G♯-B-D-F | 小九度 | D₉（或 D₉°） |

> 标记本课统一用 D₉，不区分大/小属九。自己写时知道是否降九音即可。

### 2. 为什么只用原位

| 原因 | 说明 |
|------|------|
| 五音构成 → 功能模糊 | 音越多越不清晰，只能靠低音定功能 |
| 低音是根音（属音） | 才能明确表达属功能 |
| 转位无通用命名 | 几乎不用转位 → 没产生约定俗成的名字 |

> **九和弦几乎只用原位。** 低音必须是属音。

### 3. 🔑 根音与九音 ≥ 九度

```
❌ 根音 G 和九音 A 相距二度 → 糊在一起，浑浊
✅ 根音 G 和九音 A 相距九度/十六度/二十三度 → 清晰
```

> 检查方法：从低音往上看，九音和根音之间至少隔一个纯八度以上。

### 4. 四部和声中的省略

五个音 → 四个声部，必须省一个：

| 省略方案 | 频率 | 原因 |
|:---:|:---:|------|
| **省略五音** | ⭐⭐⭐ | 五音不影响性质/协和度，删了95%听不出来 |
| 省略三音 | ⭐ | 缺三音=缺大小性质，听感变柔和 |
| 省略七音 | ⭐ | 缺七音=缺七度紧张感，听感变柔和 |
| 省略根音/九音 | ❌ | 没根音=不是属九；没九音=不是九和弦 |

> 模式一不受四声部限制，可写满五个音。超出五个 → 重复根音/五音。

### 5. 前面接什么

完全参考原位属七 → 主要用在**全终止**和**阻碍终止**处：

| 前面接 | 频率 |
|------|:---:|
| K₄₆ | ⭐⭐⭐（全终止标配） |
| S / S₆ / SⅡ₇ 及其转位 | ⭐⭐⭐ |
| T / T₆ | ⭐⭐ |
| TSⅥ | ⭐ |
| DVII₇ / DTⅢ | 极罕见（都是属功能，缺功能对比） |

连接方法：优先平稳 → 配完立刻三步检查。

**举例：K₄₆ → D₉（省略五音）**

```
旋律：do → si
K₄₆: C-G-C-E（低音 G）
D₉ : G-A-F-B（低音 G，省略 D）
→ 各声部平稳，无错误
```

### 6. 🔑 后面接什么（重点）

**D₉ 几乎只能接 T（主和弦），不能直接接 TSⅥ。**

#### 完整五声部 D₉ → T

```
G-B-D-F-A → C-E-G（五个音去往三个音）

固定进行：
  G → C  低音属→主
  B → C  导音→主音
  F → E  七音↓级进解决
  A → G  九音↓级进解决
  D → ？ 问题在这！
```

**平行五度问题**：D→C + A→G = 平行纯五度 ❌

→ 解法：D→**E**（就近去三级音，T 重复三音）
→ 或：A→A♭（降六级音 = 减五度→纯五度，可以）

#### 最常见：省略五音的 D₉ → T

```
G-B-F-A → C-E-E-G  （省略 D，无需处理平行五度问题）

G → C  低音属→主
B → C  导音→主音
F → E  七音↓解决
A → G  九音↓解决
= C-E-C-G（两根、一三、一五 = 重复根音的 T）
```

**→ 最常见也最简单！就是重复根音的正常 T。**

#### 省略七音的 D₉ → T

```
G-B-D-A → C-E-G

A → G  九音必须↓解决
G → C  低音属→主
D → ？ 如果在 A 上方 → D→C 是平四度（OK）；如果在 A 下方 → D→E（避免平五）
B → E  剩下的导音跳进到三音
```

#### 省略三音的 D₉ → T

```
G-D-F-A → C-E-G

A → G  九音必须↓解决
F → E  七音必须↓解决
G → C  低音属→主
D → C → D-A+G-C = 平行五度❌ → D→E（T 重复三音）
```

**特殊用法**（仅省略三音 + 九音在高声部）：九音 A 可**上跳三度→C**（不按↓级进），形成「la→do」旋律。

### 7. D₉ 可与 D₃/D₇ 自由转换

全终止/阻碍终止处，属功能内部可以串联：

```
K₄₆ → D₃ → D₇ → D₉ → T
K₄₆ → D₉ → D₇ → T
SⅡ⁵₆ → D₉ → D₇ → TSⅥ  （阻碍终止，注意 D₉ 后面必须跟 D₇ 才接 TSⅥ）
```

> K₄₆ 后面的一串 D₃/D₇/D₉ 当作**整体**看待——总时值不强于 K₄₆，首次出现的属和弦不强于 K₄₆ 的首次出现。

### 8. D₉ 自身同和弦转换

省略五音/三音/七音三种形式可互相转换，每次转换都检查：**根音与九音 ≥ 九度**。

## 二、下属九和弦

### 大调：SⅡ₉（不是 S₉！）

| 为什么是二级不是四级 |
|------|
| 四级七和弦（F-A-C-E）根音到七音是**大七度**（刺耳，古典时期几乎不用） |
| 七和弦都不用 → 九和弦更不用说 |
| 二级七和弦（D-F-A-C）根音到七音是小七度 → 经常用 → 可再叠九音 |

**C大调 SⅡ₉：D-F-A-C-E**

九音 E 也是六级音 → 可降（和声大调）：D-F-A♭-C-E 或 D-F-A♭-C-E♭ 皆可。

### 小调：几乎不用

| 小调二级九 | 小调四级九 |
|------|------|
| B-D-F-A-C：根三五=减三 + 根-九=小九度 → 极度不协和 | D-F-A-C-E：结构上可选，但有争议（有些教材不列入） |
| 极罕见 | 有人用，本课不强求 |

### 要点（完全参考属九）

| 要点 | 内容 |
|------|------|
| 只用原位 | 低音 = 根音（二级音）才能明确下属功能 |
| 根音-九音 ≥ 九度 | 同属九 |
| 省略五音为主 | 同属九 |
| 前后接什么 | 参考 SⅡ₇：接属功能→七音/九音↓解决；接主功能→横向保持 |
| 自身可转换 | 与 SⅡ₃/SⅡ₇ 自由串联 |

## 三、属九 vs 下属九 速查对比

| | 属九 D₉ | 下属九 SⅡ₉ |
|------|:---:|:---:|
| 根音 | 五级音（属音） | 二级音 |
| 结构（C大调） | G-B-D-F-A | D-F-A-C-E |
| 是否有六级音 | 有（A = la） | 有（E = mi，不是六级） |
| 和声大调降音 | 降九音 A→A♭ | 降三音 F→A♭（即降六级音，变 sⅡ₉） |
| 主要用于 | 全终止 / 阻碍终止 | 同 SⅡ₇ 的位置 |
| 省略哪个音 | 五音（D） | 五音（A） |
| 后面接 | T（主） | D / D₇ / K₄₆ |

---

## 四、低音-和弦对照表新增

| 低音 | 新增加和弦 |
|------|------|
| V 级音（C大调=G） | **D₉** |
| II 级音（C大调=D） | **SⅡ₉** |

## 练习要求

- **模式零**：高音题 + 低音题各一道，尽量练习 D₉ 和 SⅡ₉
- **模式一**：一道旋律，柱式/分解/混合各配一个
  - 可写满五个音（不用省略）
  - 可用 D₉ 和 SⅡ₉ 丰富和声色彩
