# 【和声】第40课 — 导六和弦、导七和弦与其转位

> 讲师：西蒙（三分钟音乐社）
> 来源：B站 BV1t1WpzWEWs
> 提取时间：2026-07-14

## 本课定位

🎯 回到模式零，学习调式最后一个和弦——VII 级导和弦。这是调内七个和弦里**最不协和**的一个，用法也最特殊。核心记住：**导七→主时，主和弦优先重复三音**。

---

## 一、结构与听觉

### 为什么最不协和

| 和弦级数 | I | II | III | IV | V | VI | **VII** |
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| 根三五音类型 | 大三 | 小三 | 小三 | 大三 | 大三 | 小三 | **减三** |

> 其他六级都是大三或小三（协和），唯独 VII 级是减三和弦 → 极度不协和、紧张。

### 构成

```
C大调：VII₇ = B-D-F-A （减小七和弦）
a小调：VII₇ = G♯-B-D-F （减减七和弦）
```

---

## 二、功能与标记

### 属功能组

VII 与 V₇ 高度相似（共有 B-D-F 三个音，仅差 G vs A）：

| V₇（属七） | VII₇（导七） |
|:---:|:---:|
| G-B-D-F | B-D-F-A |
| 属功能 | **属功能** |

→ 标记统一用 **D** 开头，加罗马数字 VII 区分。

### 完整标记体系

| 形式 | 标记 |
|------|------|
| 导三原位 | DVII（几乎不用） |
| 导六（一转） | DVII₆ |
| 导四六（二转） | DVII₄₆（几乎不用） |
| **导七原位** | DVII₇ |
| **导五六（一转）** | DVII⁵₆ |
| **导三四（二转）** | DVII³₄ |
| **导二（三转）** | DVII₂ |

### 两种结构的区分（重要！）

| 调式 | 结构 | 性质 | 标记加注 |
|------|------|------|:---:|
| **自然大调** | B-D-F-A | 减小七和弦 | DVII₇ |
| **和声大调**（降VI级） | B-D-F-A♭ | 减减七和弦 | DVII₇° |
| **和声小调** | G♯-B-D-F | 减减七和弦 | Dvii₇° |

> 标记规律：D 大写、大调用大写 VII、小调用小写 vii、减减结构加小圈 °。

**用法完全相同**，区别仅在结构导致的连法细节（见第四点）。

---

## 三、前后连接法（宏观）

### 前面可以接什么

| 前面接 | 频率 | 说明 |
|------|:---:|------|
| IV（S） | ⭐⭐⭐ | 下属→属，紧张感层层推进 |
| II（SⅡ） | ⭐⭐⭐ | 同上 |
| I（T） | ⭐⭐ | 主→属，功能鲜明对比 |
| V（D） | ⭐⭐ | 属功能内部转换，延长紧张感 |
| VI（TSⅥ） | ⭐ | 尽量少用 |
| **K₄₆** | ❌ | **禁止**！K₄₆ 只绑定纯属和弦（需有属音），导和弦里没有属音 |

### 后面可以接什么

| 后面接 | 频率 | 说明 |
|------|:---:|------|
| **I（T/T₆）** | ⭐⭐⭐ | 解决紧张感，**优先重复三音** |
| V（D/D₇） | ⭐⭐ | 属功能内部转换，延续紧张感 |
| VI（TSⅥ） | ⭐ | 极罕见（两头不讨好：既没解决也没延续） |
| IV/II | ❌ | 逆功能进行 |

## 四、导七 → 主和弦：本课最重点

### 核心规则

> 🚨 **导七与其转位接一级主和弦时，一级优先重复三音。**

### 为什么必须重复三音

**以自然大调 C: B-D-F-A → C-E-G 为例：**

1. 七音 A ↓ 必须→ G（七音解决铁律，不可改）
2. B（导音）→ C、F（四级音）→ E、D（二级音）→ C（按不稳定音倾向就近解决）
3. 结果：**D→C + A→G = 平行纯五度！❌**

→ 必须把 D→C 改为 **D→E**（退而求其次，E 也是稳定音）
→ 主和弦变成 **C-E-E-G**（重复三音）

```
正确解法：
  B → C  （导音→主音）
  D → E  （二级音→三音，避免平五）
  F → E  （四级音→三音）
  A → G  （七音必须↓解决）
  = C-E-E-G（T 重复三音）
```

### 两种结构的小区别

| 结构 | 自身有纯五度？ | 平行五度风险 | 不重复三音可以吗？ |
|------|:---:|:---:|:---:|
| 减小七（自然大调）| ✅ 有（D-A） | ⚠️ 需警惕 | 不犯错误时可以，但**仍不优先** |
| 减减七（和声大调/小调）| ❌ 无（仅减五度） | 无 | 可以，但**仍不优先** |

> 无论哪种结构，「重复三音优先」已成长期形成的用法习惯。

### 各转位接主和弦对照

| 导和弦 | 后面接 | 关键 |
|------|:---:|------|
| DVII₇ | T（重复三音） | 注意平五，D→E |
| DVII⁵₆（自然大调）| **只能接 T₆**（不能接 T） | 低音 D→C 必然引发平行五度 |
| DVII⁵₆（减减结构）| T 或 T₆，T₆ 优先 | 无平五风险 |
| DVII³₄ | **T₆**（重复三音优先） | 低音 F→E |
| DVII₂ | T₄₆（仅经过组合）/ K₄₆（极罕见）| 低音 A→G 后只能是 so，不构成 T 或 T₆ |

## 五、导七 → 属和弦（简单）

导七和属七共有 B-D-F → 几乎只需改一个音：

```
最佳平稳连接：
  DVII₇  → D⁵₆     B-D-F-A → B-D-F-G （只改 A→G）
  DVII⁵₆ → D³₄     D-F-A-B → D-F-G-B
  DVII³₄ → D₂      F-A-B-D → F-G-B-D
  DVII₂  → D₇      A-B-D-F → G-B-D-F
```

> 保持 B-D-F 不动，A ↓→ G。都是属功能内部转换，用来**延长属功能紧张感**。

---

## 六、导三四（DVII³₄）的特殊身份

DVII³₄ 的低音是下属音（如 C 大调的低音=F），且把 B 遮住就是 SⅡ⁵₆（F-A-D）：

→ **具有相当程度的下属功能**

### 两个额外用法

| 用法 | 说明 |
|------|------|
| **引向 K₄₆** | DVII³₄ → K₄₆ → D → T，此处的 DVII³₄ 体现下属功能（K₄₆ 前面不宜用纯属功能） |
| **变格补充终止** | 全终止后 DVII³₄ → T（替代 S 的位置） |

---

## 七、导三和弦与其转位（第五点）

### 为什么不用原位导三

| 原因 | 说明 |
|------|------|
| ① 重复根音 → 纵向上必然出现**两处**增四/减五 | 减三和弦本性 |
| ② 低音必然参与不协和音程 | 低音 + 减五度 → 放大不协和感 |

→ **结论：不使用原位 DVII，最多用重复三音的 DVII₆。**

### DVII₆ 的两种主要用法

| 用法 | 例子 | 说明 |
|------|------|------|
| **经过和弦** | T → DVII₆ → T₆ 或反向 | 各声部非常平稳 |
| **配 la-si-do 旋律** | S → DVII₆ → T/T₆ | 这三个音的和弦选择比较固定 |

### 小调额外注意点

小调 VI→VII→I（如 a小调 F→G♯→A）是**增二度**，不好听 → 把 F 变成 **F♯**（大 S₆）：

```
a小调：F♯ → G♯ → A  配 大S₆ → Dvii₆ → t/t₆
```

> 与第14课「低音增二度用大 S₆」同一技巧，本课扩展到上方三声部均可使用。

### 导四六（DVII₄₆）

**几乎不使用**。目前没有适合的经过/辅助搭配。

## 八、新增经过与辅助和弦组合

导和弦加入后的新经过/辅助组合（查表用）：

```
经过类（低音级进）：
  T → DVII₆ → T₆ （及反向）
  T₆ → DVII³₄ → T₄₆ → DVII⁵₆ → ... → DVII₇
  等

辅助类（低音保持）：
  略（见完整总结表）
```

> ⚠️ 标注「注意屏幕」的组合：存在 A→G 与 D→C 两个声部，必须把 A→G 放在 D→C **下方**，避免平行五度。或用和声大调降VI级（A→A♭）消除风险。

## 十、本课核心要点速记

| 序号 | 要点 |
|:---:|------|
| ① | VII 级=导和弦，减三和弦根底，属功能组，和声里最不协和 |
| ② | 两种结构：减小七（自然大调）vs 减减七（和声大调/小调），用法相同 |
| ③ | 前面主要接 I/IV/II/V，禁止 K₄₆；后面主要接 I/V，极少接 VI |
| ④ | 🔑 **导七→主时，一级优先重复三音**（全课最重要的一句话） |
| ⑤ | 导七→属七：几乎只变一个音（A→G），极平稳 |
| ⑥ | DVII³₄ 具有下属功能 → 可引 K₄₆、可做变格补充终止 |
| ⑦ | 不用原位导三，最多用重复三音的 DVII₆ |
| ⑧ | 模式一里也可以用导七和弦了 |

## 练习要求

- **模式零**：高音题 + 低音题各一道，尽量多试导七和弦及其转位
- **模式一**：一道旋律，柱式/分解/混合各配一个，可尝试使用导七和弦
