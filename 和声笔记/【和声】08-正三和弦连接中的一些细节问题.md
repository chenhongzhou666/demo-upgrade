## 一、判断调式的补充

### 核心修正

> **结束音/起始音不一定是主音（一级音），只要是主和弦内部的音（一级、三级、五级）都有可能出现！**

第7课为了简化，给的例子全是结束在主音上的。但实际上结束在**三级音**或**五级音**也很常见，因为它们都属于主和弦（T）。

### 更新后的三步判断法

| 步骤 | 看什么 | 说明 |
|------|--------|------|
| 1 | 调号 | 得出平行大小调候选 |
| 2 | 结束音/起始音 | **只要是主和弦内部音（I/III/V 级）即可**，不一定是主音 |
| 3 | 小调升VII级 | 如果是小调，旋律中应有升VII级，不应有还原VII级 |

### 例1：结束在三级音

- 调号 2# → D大调 或 B小调
- 起始音 la(A)，结束音 #F（升发）= **D大调主和弦的三级音**
- B小调主和弦 (B-D-#F) 里没有 la(A) → 排除
- D大调主和弦 (D-#F-A) 包含 la 和升发 ✅

### 例2：大小调主和弦都有同一个音

- 调号 3b → 降E大调或C小调
- 起始终止音都是 降E → **降E大调的主音**，但也是**C小调主和弦的三级音**
- 此时第2步无法区分！必须依赖第3步：
- C和声小调需要升VII级 → 还原B（♮B），而旋律中确实有还原B
- 降E大调的VII级是降B → 不会出现还原B
- ✅ **C小调**（不是降E大调！）

### ⚠️ 易错提醒

小调的升VII级不一定带「升号」。如果自然小调的VII级原本就是降号，那「升高」后就是**还原号**。比如：
- C自然小调 VII级 = 降B → C和声小调 VII级 = 还原B（♮B）

## 二、TSD 连接的易错点

### 核心原理

当旋律的高声部和低声部（根音位置）恰好形成平行/反向八度或五度时，整个和弦连接就**直接报废**——因为平行/反向五八度适用于**任意两个声部**，中间声部怎么写都没用。

### 必须避开的配法（用 C大调举例）

#### 平行/反向八度类

| 旋律进行 | ❌ 错误配法 | 为什么错 | ✅ 正确配法 |
|----------|------------|---------|------------|
| do → fa / fa → do | do=T, fa=S | 低声部+高声部 = 平行/反向八度 | 都配 S |
| do → sol / sol → do | do=T, sol=D | 低声部+高声部 = 平行/反向八度 | 都配 T |
| fa → sol | fa=S, sol=D | 低声部+高声部 = 平行/反向八度 | fa=S, sol=T |

#### 平行/反向五度类

| 旋律进行 | ❌ 错误配法 | 为什么错 | ✅ 正确配法 |
|----------|------------|---------|------------|
| do → sol / sol → do | do=S, sol=T | 低声部=fa→do / do→sol = 平行/反向五度 | 都配 T |
| re → sol / sol → re | re=D, sol=T | 低声部=sol→do / do→sol = 平行/反向五度 | 都配 D |
| do → re / re → do | do=S, re=D | 低声部=fa→sol / sol→fa = 平行/反向五度 | do=T, re=D |

### 排除法万能

碰到以上相邻音，**用排除法自己推**：
1. 列出每个音可配的和弦
2. 排除会产生平行/反向五八度的组合
3. 剩下的就是正确答案

> ⚠️ 这些归纳只适用于**当前只学了 TSD 三个和弦**的阶段。以后和弦多了，选择就多了，这条路不通可以走别的路。所以**不要死记这个表**！

## 三、终止处 T 可省略五音

### 常规形式 vs 省略形式

| | 根音 | 三音 | 五音 |
|--|------|------|------|
| 常规 T | 2个 | 1个 | 1个 |
| 终止省略 T | **3个** | 1个 | **0个** |

### 为什么？

D → T 时，**七级导音 → 一级主音**的解决感极其重要。省略五音是为了让导音能顺利解决到主音。

**优先级排序**：
```
终止处：导音→主音  >  包含完整的五音
进行中：包含完整的五音（没必要非做省略）
```

### 适用条件

- **只在终止处（最后的 T）** 考虑省略五音
- 音乐进行中的 T 不省略，保持 2根+1三+1五 完整形式
- 如果做省略形式反而导致不平稳/平行五八度等问题 → **别强求，用常规形式**

### 示例（C大调结尾）

```
旋律: ... → re → do

省略五音版 (更好 ✅):
D: G  B  D  B     →  T: C  C  E  C
   (低声部G-C)         (导音B→主音C)

常规版 (也可以 ✅):
D: G  B  D  B     →  T: C  G  E  C
   (同样正确)          (导音B→五音G，解决不彻底)
```

### 判断流程

```
1. 尝试做 3根音+1三音 的省略形式
2. 检查是否违反7条规则（不平稳、平行五八度等）
   ├── 不违反 → 用省略形式（更好）
   └── 违反 → 用常规 2根+1三+1五 形式
```
