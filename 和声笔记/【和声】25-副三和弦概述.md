---

## 一、正三和弦 vs 副三和弦

| | 正三和弦（已学） | 副三和弦（新） |
|------|:---:|:---:|
| 级数 | Ⅰ、Ⅳ、Ⅴ | **Ⅱ、Ⅲ、Ⅵ、Ⅶ** |
| 功能 | 纯粹的 T、S、D | 不纯粹，是 T/S/D 的混合或弱化版 |
| 标记 | T、S、D | 需加级数区分（如 SⅡ、DⅦ 等） |

> 一个调有七个音，对应七个三和弦。学完副三和弦后，调内七个和弦就全都掌握了。

---

## 二、副三和弦的功能分组（核心理论）

### 推导逻辑：与正三和弦有两个共同音

**主功能组**：Ⅰ 级的构成音 = do-mi-sol

| 级数 | 和弦 | 与Ⅰ级的共同音 | 纯正度 |
|:---:|------|:---:|:---:|
| **Ⅰ** | do-mi-sol（T） | 三个全有 | ⭐⭐⭐ 最纯正 |
| **Ⅵ** | la-do-mi（TSⅥ） | do + mi | ⭐⭐ 较接近 |
| **Ⅲ** | mi-sol-ti（DTⅢ） | mi + sol | ⭐⭐ 较接近 |

> Ⅰ、Ⅵ、Ⅲ 统称 **主功能组**。Ⅵ 和 Ⅲ 各有两个构成音与 T 相同，是对主功能的「微微调整」。

**下属功能组**：Ⅳ 级的构成音 = fa-la-do

| 级数 | 和弦 | 与Ⅳ级的共同音 | 纯正度 |
|:---:|------|:---:|:---:|
| **Ⅳ** | fa-la-do（S） | 三个全有 | ⭐⭐⭐ 最纯正 |
| **Ⅱ** | re-fa-la（SⅡ） | fa + la | ⭐⭐ 较接近 |
| **Ⅵ** | la-do-mi（TSⅥ） | la + do | ⭐⭐ 较接近 |

> Ⅳ、Ⅱ、Ⅵ 统称 **下属功能组**。Ⅵ 同时属于两个组！

**属功能组**：Ⅴ 级的构成音 = sol-ti-re

| 级数 | 和弦 | 与Ⅴ级的共同音 | 纯正度 |
|:---:|------|:---:|:---:|
| **Ⅴ** | sol-ti-re（D） | 三个全有 | ⭐⭐⭐ 最纯正 |
| **Ⅲ** | mi-sol-ti（DTⅢ） | sol + ti | ⭐⭐ 较接近 |
| **Ⅶ** | ti-re-fa（DⅦ） | ti + re | ⭐⭐ 较接近 |

> Ⅴ、Ⅲ、Ⅶ 统称 **属功能组**。Ⅲ 也同时属于两个组！

## 三、各副三和弦的标记

| 级数 | 功能归属 | 标记 | 说明 |
|:---:|------|:---:|------|
| **Ⅱ** | 仅下属功能 | **SⅡ** | 最单纯的下属副和弦 |
| **Ⅶ** | 仅属功能 | **DⅦ** | 最单纯的属副和弦 |
| **Ⅵ** | 主+下属（复合） | **TSⅥ** | 偏主功能（含主音 do），T 写在前面 |
| **Ⅲ** | 属+主（复合） | **DTⅢ** | 偏属功能（含属音 sol），D 写在前面 |

### 标记命名规则

- 标记字母代表功能归属（T/S/D）
- 需要加级数的加在后面（SⅡ vs S₄）
- 复合功能的按**偏向哪个**决定谁在前面
  - TSⅥ：偏 T（含主音 do 但不含下属音 fa），T 在前
  - DTⅢ：偏 D（含属音 sol 但不含主音 do），D 在前

---

## 四、复合功能的理解（三原色比喻）

西蒙用 **红绿蓝三原色** 比喻 T/S/D：

| 比喻 | 功能 |
|:---:|------|
| 红 | T（主功能） |
| 绿 | S（下属功能） |
| 蓝 | D（属功能） |

- **TSⅥ**（六级）= 红+绿 = **黄色**——既可用于需要「红」的场景（如 D 后面解决→偏 T），也可用于需要「绿」的场景（如 T 与 D 之间推进→偏 S）
- **DTⅢ**（三级）= 蓝+红 = **紫色**——既偏 D 又偏 T，同理

### 核心认知

> 不需要非要去纠结 TSⅥ 到底「偏 T」还是「偏 S」。**重要的是记住它能在哪些地方用。** 就像黄色可以画太阳（代替红色）也可以画麦田（代替绿色）——记住用法比纠结归类重要得多。

**典型例子**：
- T→S→D→**TSⅥ**：D 后面用 TSⅥ 替代 T → TSⅥ 扮演了「主功能」角色
- T→**TSⅥ**→D→T：T 和 D 之间用 TSⅥ → TSⅥ 扮演了「下属功能」角色

## 五、功能组全景图

```
        主功能组 (T)
       /    |    \
      Ⅰ    Ⅵ    Ⅲ
      T   TSⅥ  DTⅢ
     /纯  /复合 /复合 \
    /正  /有T+S /有D+T \
   /    /       /       \
  下属功能组 (S)        属功能组 (D)
   |    |    |           |    |    |
   Ⅳ    Ⅱ    Ⅵ          Ⅴ    Ⅲ    Ⅶ
   S   SⅡ   TSⅥ        D   DTⅢ  DⅦ

   注：Ⅵ = TSⅥ 同时属于 T 组和 S 组（复合功能）
       Ⅲ = DTⅢ 同时属于 T 组和 D 组（复合功能）
```
