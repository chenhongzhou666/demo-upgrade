## 一、终止式的命名细化

回忆第3课的三类进行：

| 进行类型 | 定义 |
|---------|------|
| **正格进行** | T ↔ D |
| **变格进行** | T ↔ S |
| **完全进行** | T → S → D → T（三种功能全涉及） |

### 半终止的命名

| 名称 | 结束在 | 优先级 |
|------|--------|--------|
| **正格半终止** | D（属和弦） | ⭐⭐⭐ 优先 |
| **变格半终止** | S（下属和弦） | ⭐ 做不了 D 才考虑 |

> 优先正格半终止，因为 D 天生倾向引导下一句的主功能。

### 全终止的命名

| 名称 | 结构 | 优先级 |
|------|------|--------|
| **完全终止** | S → D → T | ⭐⭐⭐ 最优 |
| **正格全终止** | D → T（如 T → D → T） | ⭐⭐ 次选 |
| **变格全终止** | S → T | ⭐ 前两种都做不了才用 |

> 全终止优先做完全终止（S→D→T），层层递进，紧张→释放的舒适感最强。

**重要提醒**：这里的 T/S/D 代表的是**功能**，不只是原位三和弦。S₆→T 也叫变格终止，S→D₆→T 也叫完全终止。以后学了更多下属类/属类和弦后都可等效替换。

## 二、完满终止 vs 不完满终止

### 三个条件（全部满足 = 完满终止）

| 条件 | 内容 | 影响程度 |
|------|------|---------|
| **条件一** | 最后的 T 和它前面的 S 或 D 都必须是**原位** | ⭐⭐⭐ 非常关键 |
| **条件二** | 最后的 T 必须在**强拍或次强拍** | ⭐⭐⭐ 非常关键 |
| **条件三** | 最后的 T 的**旋律声部必须是根音**（主音） | ⭐⭐ 锦上添花 |

> **完满终止** = 三个条件全部满足 → 终止感极强，彻底的句号
> **不完满终止** = 任意一个条件不满足 → 终止感被削弱

### 听觉对比（西蒙用控制变量法演示）

```
完满终止：原位D→原位T，T在强拍，旋律在根音   → 稳稳到家 ✅
不完满①：D→T₆（条件一不满足）               → 回家感打折扣
不完满②：D₆→T（条件一不满足）               → 略弱但新手可能听不出
不完满③：T在弱拍（条件二不满足）             → 戛然而止，仓促感明显 ⚠️
不完满④：旋律结束在三音（条件三不满足）       → 不够干脆彻底
三个都不满足                                  → 根本不叫终止
```

### 实操优先级

| 场景 | 做法 |
|------|------|
| **配和声练习**（旋律固定） | 条件三不用管（旋律改不了），但条件一和二要尽量满足 |
| **自己创作** | 看需要：要强终止感→三条件全满足；要留余地→旋律结束在三音/五音 |
| **全终止处** | 优先原位 D→原位 T，T 在强拍或次强拍 |

## 三、第一乐句也可以用 T（不完满形式）

### 核心逻辑

> **第二乐句的终止感必须大于第一乐句**，才能形成「先逗号，后句号」的正常递进。

### 两种合法的平行乐段终止模式

| 模式 | 第一乐句（半终止） | 第二乐句（全终止） | 典型例子 |
|------|-------------------|-------------------|---------|
| **模式一**（上节课） | D（属和弦） | T（可完满可不完满） | 大量练习和作品 |
| **模式二**（本节课新） | 不完满 T | 完满 T | 致爱丽丝、孙燕姿《遇见》、莫扎特作品 |

**模式二的逻辑**：
- 第一乐句结束在 T，但用不完满形式（通常是**三音旋律位置**）
- 因为不完满 = 终止感被削弱 = 仍像逗号
- 第二乐句用完满 T = 终止感更强 = 句号
- 同样是「先逗号后句号」的递进逻辑

### 为什么不通过其他方式做不完满？

| 尝试方式 | 为什么不行 |
|---------|-----------|
| 把 T 放弱拍（改条件二） | 影响整体流畅度，听觉不顺 ⚠️ |
| 用 T₆ 代替 T（改条件一） | 断句感/终止感不够，听起来没完 |
| 用五音旋律位置 | 旋律是五音（= 属音）时，配 D 比配不完满 T 更好，没必要强行配 T |

> **最常见的不完满 T 形式：三音旋律位置**（条件三不满足）

## 四、乐段的更多类型

### 按乐句数量分类

| 类型 | 说明 | 常见度 |
|------|------|--------|
| **单乐句乐段** | 一气呵成，中间找不到分割点 | 较少（练习题中会出现） |
| **两乐句乐段** | 两个乐句，最常见 | ⭐⭐⭐ 绝对主流 |
| 三乐句/四乐句乐段 | 更多乐句 | 较少，暂不展开 |

### 平行乐段

**定义**：两个乐句「同头变尾」的乐段。

**「同头」不要求完全一模一样**：
- 旋律、节奏、和声**至少有一方面雷同**即可
- 可用**模进**（后一片段对前一片段重复但音高有变化，如 do-re-mi → mi-fa-sol，高三度）
- 关键在于不会让人觉得第二乐句是「另一个中心思想」

### 平行乐段的两种终止模式

| | 第一乐句 | 第二乐句 | 第二乐句 T 的要求 |
|------|---------|---------|------------------|
| **上面那种** | D | T | 可完满可不完满（因为 D 本身终止感极弱，T 怎么都比 D 强） |
| **下面那种** | 不完满 T | T | **必须用完满 T**（因为第一句已有一定终止感，第二句必须更强） |

## 五、实战工作流（拿到旋律后的三步）

### 第一步：判断调性
快速确定大调/小调，写好 TSD 参考音。

### 第二步：判断结构
- 8 小节 → 看第 4 小节、16 小节 → 看第 8 小节，找可能的分割点
- 检查两点：①能不能配 D（不强求）②两个乐句是否「同头变尾」
- 满足 → **平行乐段**（有半终止）
- 实在找不到 → **单乐句乐段**（无半终止，正常配即可）

### 第三步：优先布局终止式的和弦

| 位置 | 优先配法 |
|------|---------|
| **半终止**（如有） | 优先 D（正格半终止） |
| **全终止**（最后） | 优先 S→D→T（完全终止），且最后 D 和 T 都原位、T 在强拍/次强拍 |

> 先标记好终止处的和弦 → 再从头开始按正常流程配和声 → 这样就做到了**和声布局的宏观控制**。

## 六、本课总结

| 新概念 | 一句话 |
|--------|--------|
| **正格/变格/完全终止** | 按涉及 D/S/T 的哪个来命名 |
| **全终止优先级** | 完全(S→D→T) > 正格(D→T) > 变格(S→T) |
| **完满终止三条件** | 最后两个和弦原位 + T 在强拍 + 旋律在根音 |
| **不完满 T 的典型用法** | 第一乐句终止用三音旋律位置的 T |
| **平行乐段** | 同头变尾的两乐句乐段，最常见 |
| **终止感递进** | 第二句的终止感必须 > 第一句（先逗号后句号） |

### 终止式速查表

```
              ┌─ 半终止 ─ 正格半终止 (D) ← 优先
              │         └ 变格半终止 (S) ← 备选
终止式 ──────┤
              │         ┌ 完全终止 (S→D→T) ← 最优
              └ 全终止 ─ 正格全终止 (D→T)   ← 次选
                        └ 变格全终止 (S→T)   ← 末选
              ┌ 完满：两和弦原位 + T强拍 + 旋律根音
              └ 不完满：任一条件不满足
```

## 七、重要提醒

- 本课讲的仍是**最常见情况**，终止式还有更多类型（补充终止、阻碍终止、侵入终止等），以后遇到再讲
- 乐段结束在 D 或 S 上也是存在的（目的是制造期待，引导下一个乐段），但目前练习范围是单个乐段，暂不涉及
- 贝多芬作品中出现的「平行八度」实际上是单旋律叠加八度做加强效果，不是四部和声练习中的平行八度错误，以后会专门讲
