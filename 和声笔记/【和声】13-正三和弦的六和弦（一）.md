## 一、「六和弦」的两种意义

> 乐理课学过但必须复习清楚，否则交流极易出错。

| 意义 | 定义 | 使用场景 |
|------|------|---------|
| **意义一**（本课程采用） | 三和弦的**第一转位**，三音在低音 | 形容和弦功能/结构 |
| **意义二**（本课程不用） | 三和弦附加大六度音（如 CEGA） | 流行音乐标记与交流 |

例如 CEG 和弦，把三音 E 放在低音 → C/E，这就是意义一的「六和弦」。「六」来源于：在第一转位的狭义形态中，三音与根音距离为六度。

> 本课程说的「六和弦」一律 = 三和弦的第一转位。

## 二、六和弦的标记

| 和弦 | 原位标记 | 六和弦标记 |
|------|---------|-----------|
| 主和弦 | T | **T₆** |
| 下属和弦 | S | **S₆** |
| 属和弦 | D | **D₆** |
| 小调同理 | t / s / D | **t₆ / s₆ / D₆** |

标记中的「6」写在 T/S/D 的右下方（也可右上方、正右方，不同教材不同，不必纠结）。

## 三、六和弦的重复音规则

### 核心结论

> **六和弦上方三声部：要么重复根音，要么重复五音，都很常见。一般不重复三音。**

### 为什么一般不重复三音？

1. **声学原因**（同第4课）：三度及其倍数频率出现过多会降低协和度；三音只需出现一次就足够决定和弦的大小性质
2. **六和弦中三音已经在低声部**（关键位置），上方再重复三音听感会差
3. 少数特殊情况下三音也可以重复——第14课细讲

### 为什么重复根音和五音都可以？

这和**功能纯正度**有关（见下文）。根音或五音都可以，**关键看怎么和前后和弦连接更平稳、不出错**。

### 实操心法

> **不要先想「我要重复根音还是五音」再配，而是先做最平稳的连接，配出来重复谁就是谁。**

两个例子（C大调 T→S₆）：

```
例1：高音do→do（保持），中声部mi→fa，次中do→do（保持）
     → S₆ 上方 = 两个五音 + 一个根音（重复五音）

例2：高音mi→fa（上二度），中声部do→do（保持），次中sol→do（下五度）
     → S₆ 上方 = 两个根音 + 一个五音（重复根音）
```

两个都对，因为都是「最平稳连接」自然产生的结果。

## 四、核心概念：功能纯正度（本课最重要的理论）

### 原位 = 功能最纯正

- **T 的原位**：主音 C 在低音 → 最纯正的主功能，最稳定的「回家感」
- **T₆**：三音 E 在低音 → 仍然有主功能的构成音，但**功能性不够纯正**

### 听觉对比

- 音乐结尾用**原位 T** → 稳稳到家，圆满结束
- 音乐结尾用 **T₆** → 差了一丝意思，有「还没到家」「音乐还想继续」的感觉 [需配图]

### 推论

| 和弦形态 | 功能纯正度 | 适合角色 |
|----------|-----------|---------|
| 原位 T/S/D | ⭐⭐⭐ 最纯正 | 关键结构位置（开头、结尾、需要强调功能处） |
| 六和弦 T₆/S₆/D₆ | ⭐⭐ 不够纯正 | **过渡、铺垫**性质的角色 |

> **低音可以很明显地改变一个和弦在功能性上的纯正度。** 构成音不变、无论上方怎么变换排列，都不会带来质变；但低音一变，功能性立刻就变了。

### 这解释了「为什么重复根音或五音都可以」

正因为六和弦主要做过渡铺垫，我们对它功能纯正性的要求没那么高 → 没必要非重复根音（主/下属/属音）去强调功能 → 根音和五音哪个让连接更平稳就用哪个。

## 五、混合排列

### 三种排列法回顾

| 排列法 | 判断标准（上方三声部两两之间） |
|--------|--------------------------|
| **密集排列** | 全部 ≤ 四度 |
| **开放排列** | 全部 ≥ 五度 |
| **混合排列** | 既有 ≤ 四度，又有 ≥ 五度 |

### 六和弦与混合排列

- 六和弦**很容易**形成混合排列（但也可以形成密集/开放排列）
- 原位 TSD 只会出现密集或开放排列，所以混合排列当时没讲

### 实操态度

> 这三个词只是描述和弦纵向造型的名词概念，**配和声时不需要刻意去想**。你只需要关注：上方三声部不超过纯八度、声部尽量平稳、不交叉不超越、避免五八度问题即可。

## 六、本课总结

| 要点 | 一句话 |
|------|--------|
| 六和弦定义 | 三和弦的第一转位，三音在低音 |
| 标记 | T₆ / S₆ / D₆（数字写在右下方） |
| 重复音 | 根音或五音都可，取决于和前后和弦连接的平稳度；一般不重复三音 |
| 功能纯正度 | 六和弦功能不够纯正 → 主要做过渡铺垫角色，不能胜任需要纯正功能的位置 |
| 混合排列 | 上方三声部既有≤四度又有≥五度；了解即可，实操不用管 |
