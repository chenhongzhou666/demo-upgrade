## 一、转位和弦的运用

### 为什么用转位

之前全用原位 → 低音线条完全由和弦根音决定 → 选择少。

**用转位 → 可以「雕刻」低音旋律线**，让伴奏的另一条隐藏旋律（低音）更好听。

### 两个方面

#### 方面一：构建有美感的低音线条

低音 = 每次换和弦时第一下弹的那个最低音。听觉上会把这些低音串成一条隐藏旋律。

```
全用原位：低音线 = do sol la mi fa do re sol（跳跃大）
加转位后：低音线 = do si la sol fa mi re sol（平稳下行）

哪个好听？没有绝对答案，但「多平稳少跳进」是安全的默认原则。
```

> 多平稳少跳进——写旋律的原则同样适用于写低音线条。

#### 方面二：特定位置的「功能性强转位」

| 位置 | 要求 | 原因 |
|------|------|------|
| **音乐结束** | 必须原位 T | T₆ 终止感不够彻底 |
| **半终止/全终止的属功能** | 尽量用属音在低音（K₄₆→D 或原位 D）| 属功能感最大化 |
| **半终止/全终止前一个和弦** | 尽量用**下属音**在低音 | 低音形成「下属音→属音」，功能推进更正统有力 |

**例子：**

```
SⅡ₇（原位，低音=D） → D（低音=G）
   ↑ 二级音→属音，没问题但不如下面的正统

SⅡ⁵₆（一转，低音=F） → D（低音=G）
   ↑ 下属音→属音！功能推进更有力
```

### 转位在模式一里的注意事项

#### ① 六和弦尽量不重复三音

```
✅ S₆: 低音=三音出现一次，其他声部只用根音+五音 → 清爽通透
❌ S₆: 左右手都弹到三音 → 听觉略浑浊，且容易形成隐藏的平行八度
```

#### ② 模式一可以用 DTⅢ 的转位

第41课说模式零不用三级转位，但模式一没有这个限制。

#### ③ 左右手冲突时的处理（四种取舍）

当不重复三音+平稳连接和「右手旋律不能改」冲突时：

| 方案 | 做法 | 缺陷 |
|:---:|------|------|
| A | 左手冲突处往下移最近音 | 连续同音=卡壳感 |
| B | 保持另几层平稳，冲突层特殊处理 | 织体形态突然不统一，可能有违和感 |
| C | 冲突处左手不弹，让给右手 | 瞬间单薄，谱面不统一 |
| D | 接受四部同向 | 所有层次同方向走（也有缺陷） |

> **没有完美解。** 分析每种方案的优劣 → 选择自己最能接受的缺陷。这就是模式一中「有理由的取舍」。

## 二、气口加花 ⭐

### 概念

**当主旋律在休止或长音符时（气口），伴奏右手跳出来弹一小段旋律（加花），让这个地方不枯燥。**

### 三个要点

#### ① 加花旋律不要写太低

音越高越明亮 → 既然是来「展示」的就别藏着掖着。低音区加花=加了个寂寞。

#### ② 加花末尾要平稳衔接后面的和弦

> **加花不是孤立的，加花 + 下一和弦的第一个音 = 一个整体旋律。**

```
✅ 加花末尾 = re → 下一和弦最高音 = mi（三度，平稳）
❌ 加花末尾 = sol → 下一和弦最高音 = mi（超过纯八度跳进，断层感）
```

#### ③ 节奏、旋律：自由尝试！

做到了①②之后，加花用什么节奏（八分/三连音/延音线/附点）和什么旋律音，**没有标准答案**。多加多听才能积累「加花语汇库」。

如果加花旋律感觉单薄 → 可以在下方叠加当前和弦的构成音（如 D 和弦索西来，在加花下方补 re）。

### 标记问题

加花的音要不要算进和弦标记？**不算。**

```
加花的本质 = 临时的小旋律装饰
和弦的本质 = 此处的和声骨干（属和弦 G-B-D）

→ 标记就标 D，不需要把加花音硬塞进标记
```

### 目前练习范围

先只在**半终止处**练气口加花。原因：
- 半终止处旋律通常是长音符（容易有气口）
- 半终止处本来就适合做变化（第37课统一与变化）

## 三、转位和弦选择自检表

写完后问自己：

| 检查项 | 问题 |
|:---:|------|
| 1 | 低音线条我会唱吗？好听吗？有没有试试其他原位/转位组合？ |
| 2 | 结束和弦是原位 T 吗？（不是的话终止感够吗？） |
| 3 | 半终止/全终止的属功能和弦，属音在低音吗？ |
| 4 | 终止前一个和弦，能不能用下属音在低音（功能推进更强）？ |
| 5 | 六和弦有没有重复三音？（没必要的重复建议去掉） |

---

## 练习要求

- **模式零**：高音题 + 低音题各一道（巩固上节课）
- **模式一**：一道旋律，柱式/分解/混合各配一个
  - 必须使用转位和弦（考虑低音线条 + 特定位置功能性）
  - 必须在半终止处做气口加花
  - 柱式/分解/混合三个版本写三种不同的加花
