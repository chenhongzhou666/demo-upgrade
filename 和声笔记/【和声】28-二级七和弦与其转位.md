## 本课定位

🎯 在 SⅡ 三和弦基础上加入七音，形成**二级七和弦家族**：SⅡ₇、SⅡ⁵₆、SⅡ³₄、SⅡ₂。因属七体系（第22-23课）已打下基础，本课原理完全相通。核心新规则只有一条：**七音要么下行级进解决，要么保持**。

---

## 一、四种形态速览（C大调为例）

| 形态 | 标记 | 低音 | 构成音 |
|:---:|:---:|:---:|------|
| 原位 | **SⅡ₇** | re（根音） | re-fa-la-do |
| 第一转位 | **SⅡ⁵₆** | fa（三音） | fa-la-do-re |
| 第二转位 | **SⅡ³₄** | la（五音） | la-do-re-fa |
| 第三转位 | **SⅡ₂** | do（七音） | do-re-fa-la |

### 大调里的两个版本

| 版本 | 性质 | 使用频率 |
|------|------|:---:|
| 正常六级音（la） | 小小七和弦 | ⭐⭐⭐ 主流 |
| 降六级音（♭la）和声大调 | 减小七/半减七 | ⭐ 偶尔1-2次 |

> 小调里只有减小七性质版本，正常使用即可。

## 二、核心要点

### ① 最优先使用 SⅡ⁵₆

原因：低音是下属音 fa → 属下功能感最强，听感最丰富（4个构成音），是四个里最常用的。

### ② 前方可接任何和弦（不逆功能即可）

与 SⅡ/SⅡ₆ 规则一致——属功能组以下属功能之前都可以。

### ③ 七音规则（本课核心 ⭐）

**任何七和弦的七音，只有两种合法归宿：**

| 后面接什么 | 七音归宿 | C大调例子 |
|------|:---:|------|
| **D / D₆ / D₇ / D₅₆ / D₃₄ / D₂** | ↓ 级进解决 | do → ti |
| **T / T₆ / T₄₆ / K₄₆** | **横向保持**（不动） | do → do |

**为什么属七只能下行不能保持？** 因为属七的七音（fa）在 T/T₆ 中不存在 → 只能下行到 mi。二级七的七音（do）在 T 中存在 → 可保持。

### ④ 同和弦转换时七音可灵活

- 同一和弦自己接自己（SⅡ₇→SⅡ₇）：七音可在声部间自由转移
- 同功能不同转位互接（SⅡ₇→SⅡ⁵₆→SⅡ³₄→SⅡ₂）：**尽量**让七音保持同一声部（精益求精）

## 三、具体连接的推理方法

西蒙反复强调：**不要死记「某和弦接某和弦的声部进行」**，而是遵循统一逻辑：

> **先保证七音归宿 → 其他声部优先平稳 → 配完立刻检查 → 没基础错误就可用**

### SⅡ₇/SⅡ⁵₆/SⅡ³₄/SⅡ₂ → 属功能和弦

**最优造型**（两个声部保持 + 两个声部级进下行）：

| 前 | → | 后 | 保持 | 级进↓ |
|:---:|:---:|:---:|------|------|
| SⅡ₇ | → | D₃₄ | re→re, fa→fa | do→ti, la→sol |
| SⅡ⁵₆ | → | D₂ | fa→fa, re→re | do→ti, la→sol |
| SⅡ³₄ | → | D₇ | la→la, fa→fa | do→ti, re→do |
| SⅡ₂ | → | D₅₆ | do→do, re→re | fa→mi, la→sol |

> 这四种造型**不用检查，一定正确**。但这不是唯一合法方式——任何配法只要七音解决/保持了+没基础错误，都可用。

### SⅡ₇/SⅡ⁵₆/SⅡ³₄ → T 或 T₆

七音保持（do→do），其他声部优先平稳。⚠️ 注意 SⅡ₇→T 容易产生平行五度（re-la → do-sol），需要调整声部或降六级（变成小SⅡ₇→T 则无平五问题）。

### SⅡ₂ 不能接 T₆ 和 K₄₆

因为 SⅡ₂ 的七音 do 在低音位置，只能 ↓ti 或保持 do，不能跳去 T₆ 的低音 mi 或 K₄₆ 的低音 sol。

## 六、总结

| 要点 | 内容 |
|------|------|
| **最优先** | SⅡ⁵₆（低音=下属音 + 4个构成音最丰富） |
| **七音铁律** | 接属功能 → ↓级进解决；接主功能 → 横向保持 |
| **最优造型** | SⅡ₇→D₃₄、SⅡ⁵₆→D₂、SⅡ³₄→D₇、SⅡ₂→D₅₆（不用检查） |
| **SⅡ₂限制** | 不接 T₆、不接 K₄₆（七音在低音，去不了） |
| **检查逻辑** | 先保证七音归宿 → 优先平稳 → 检查基础错误 → 可用 |
