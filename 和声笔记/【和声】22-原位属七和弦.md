## 本课定位

🎯 **阶段二最重要的新和弦。** 属七和弦（D₇）是属三和弦的升级版——在属三（根+三+五音）基础上加入七音，形成大小七和弦结构，带来远比属三更强的紧张感和解决渴望。这是调性和声最核心的「张力→释放」引擎。

---

## 一、「属七和弦」的两种意义

| 意义 | 定义 | 本课程采用？ |
|------|------|:---:|
| **意义一** | 调式中五级音（属音）作为根音构造的调内七和弦 | ✅ 采用 |
| **意义二** | 任何在结构上是大小七和弦的和弦 | ❌ 不采用 |

> 本课程中，属七和弦 = 调式 V₇（如 C大调 G-B-D-F，a小调 E-G#-B-D）。

---

## 二、D₇ 的六要点

### ① 标记与结构

- 标记：**D₇**（数字 7 写在右下角，表示根音上方的七度音）
- 结构：**大小七和弦**（大三和弦 + 小七度）
- 大调和小调中的属七都是大小七和弦结构

### ② 两种构成形式

| 形式 | 构成 | 特点 |
|------|------|------|
| **完整形式** | 根音 + 三音 + 五音 + 七音（四个构成音全在） | 默认形式 |
| **省略五音形式** | 根音（重复）+ 三音 + 七音 | 常用于避免平行五度，五音是 D₇ 中最不重要的音 |

> C大调 D₇：完整 = G-B-D-F；省略五音 = G-G-B-F

### ③ 排列法自由

密集、开放、混合排列都可以。唯一需要遵守的：上方三声部两两之间不超过纯八度。

### ④ 前方：D₇ 前面可以接什么

**总的思路**：先平稳连接，再检查错误。D₇ 在功能上是原位属三和弦的升级，所以前面可以接目前已学的任何和弦。

| 前方和弦 | 注意事项 |
|---------|---------|
| **T / T₆** | 注意**纯五度→减五度不是错误**（D₇ 的三音-七音形成的是减五度/三全音） |
| **S / S₆** ⚠️ | **最容易错！** S的五音→D₇的五音会形成平行五度（如 C大调 fa-do→sol-re）；应优先用**省略五音形式**，或让 D₇的五音避开 S的五音 |
| **D / D₆** | 视为**同和弦转换**（属功能内部升级，只增加一个七音），自由连接 |
| **K₄₆** | **只能在全终止处用 K₄₆→D₇**！半终止用了会太像全终止，失去逗号感 |

### ⚠️ 重点：减五度 ≠ 平行五度

属七和弦的三音与七音形成**减五度（三全音）**，这不是纯五度。之前学的禁止平行/反向/隐伏五度中的「五度」都特指**纯五度**。

- ✅ 纯五度 → 减五度 = 正确（不是错误！）
- ❌ 纯五度 → 纯五度 = 平行五度（错误）

### ⑤ 后方：D₇ → T 的解决（本课核心重点）

**核心铁律：七音必须向下级进解决到 T 的三音。** 这是最优先保证的！

#### 为什么 D₇ 比属三更紧张？

| | 属三和弦 | 属七和弦 |
|------|:---:|:---:|
| 内部音程 | 只有协和音程（纯五、大三、小三） | 含小七度（根音-七音）+ 三全音（三音-七音） |
| 不协和程度 | 仅在功能性上不稳定 | 在音程上也**不协和** |
| 解决渴望 | 强 | **非常强** |

#### D₇ → T 的三种解决方式

| 方式 | 暑期形式 | T 的形式 | 特点 | 口诀 |
|------|---------|---------|------|------|
| **①** | 完整（根三五七） | 不完整 T（缺五音） | 各音按最自然倾向解决 | 根三上五七下 |
| **②** | 完整（根三五七） | 完整 T（根三五全） | 牺牲导音→主音（导音↓到五音），**仅在导音处于内声部时可用** | — |
| **③** | 不完整（根根三七） | 完整 T（根三五全） | 最常用，平稳且 T 饱满 | — |

**方式①：根三上五七下（自然倾向）**

```
C大调：G-B-D-F → C-C-C-E
       根↑  三↑  五↓  七↓
```

每个音按自己最自然的倾向解决，但 T 缺五音。

**方式②：牺牲导音（仅内声部可用）**

```
C大调：G-B-D-F → C-E-G-C
       B不走↑到C，而走↓到G（牺牲了导音→主音的自然倾向）
```

> ⚠️ B→G（导音不解决到主音）**只能发生在内声部**。若导音在高声部，必须向上解决到主音！因为人耳对高声部最敏感。

**方式③：省略五音（最推荐）**

```
C大调：G-G-B-F → G-C-E-C
       根保持→五音，三↑，七↓
```

暑期省略五音、重复根音，解决到完整 T。这是最常用的方式。

#### 为什么省略五音后必须重复根音？

- 加强属功能（根音 = 属音）
- 重复三音/七音会产生过多不协和音程（两个三全音），且无法正常解决到 T

#### 同和弦转换时的例外

D₇ 自身做同和弦转换（连续多个 D₇）时，七音可以在上方三声部中**自由移动**，不需要级进解决。七音向下解决是 D₇→T 这个过程中的要求。

### ⑥ 实践要点

| 场景 | 要点 |
|------|------|
| **半终止** | K₄₆ 和 D₇ 只能二选一，不要同时用（太像全终止） |
| **全终止** | 鼓励用 K₄₆ → D₇ → T（最强终止感） |
| **音乐进行中**（非终止处） | 以后学了 D₇ 的转位后应少用原位 D₇，多用转位（原位 D₇→T 太像终止） |
| **旋律出现四级音**（如 C大调的 fa） | 现在可以配 D₇ 了（不止 S/S₆），但必须满足七音↓解决 |
| **低音出现五级音** | 现在可以配 D₇ 了（不止 D/K₄₆），旋律多了四级音的选择 |

## 三、D₇ 的意义总结

| 维度 | 说明 |
|------|------|
| **和声色彩** | 七音的加入让属功能从「功能性不稳定」升级到「音程也不协和」 |
| **推进力** | D₇→T 是调性和声最强的张力→释放引擎 |
| **必记铁律** | 七音必须向下级进解决到 T 的三音（D₇→T 时） |
| **省略规则** | 省略五音后必须重复根音，不能重复三音/七音 |
| **减五度** | 不是平行五度错误对象！仅纯五度受限制 |
