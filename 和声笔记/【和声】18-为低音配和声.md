## 本课定位

🎯 引入**新的练习类型：为低音配和声（低音题）**。以前的练习都是给旋律配和弦（高音题），现在反过来——给低音，让你自己创作旋律。这是旋律创作训练的起点，会贯穿整个和声课程。

---

## 一、为什么要练低音题

### 低音 ≈ 和弦已定

在我们目前学过的和弦中，**每个低音基本只对应一个和弦**：

| 低音（C大调） | 唯一对应的和弦 | 例外 |
|:---:|:---:|:---:|
| do | T | — |
| re | D₆ | — |
| mi | T₆ | — |
| fa | S | — |
| sol | **D 或 K₄₆** | 需判断终止位置 |
| la | S₆ | — |
| si | D | — |

> 低音基本已经锁定了和弦进行，我们要做的就是在这个**既定的和弦框架里创作旋律**。

### 为什么不是直接自由创作旋律？

1. 新手直接自由创作，很难把握旋律与和声、乐句乐段结构的关系
2. 可能写出来的旋律完全配不上正常的和声，也没有正常的结构
3. 先在有框架的情况下大量练习 → 慢慢领悟旋律与和声的互动规律 → 以后再自由创作

### 为什么只练高音题和低音题，不练中声部题？

两个内声部更多是**填充**角色（增加纵向饱满度），是人耳的配角。练好高低两个外声部的写作就完全够了。

## 二、低音题实战流程

### 第一步：判断调性

- 看调号 → 基本锁定大小调
- 看**最后一个低音** → 确认调性（结束音几乎都是主音）

例：0升0降 + 最后一个低音是 do → C大调

### 第二步：标注和弦 + 分析结构

- 每个低音标出对应的和弦（只有 sol 需要判断是 D 还是 K₄₆）
- 同时分析乐句乐段结构、半终止/全终止位置
- K₄₆ 只能用在半终止或全终止处

> 这一步熟练后几乎是「一眼看穿」的，因为低音已经把一切锁定了。

### 第三步：创作旋律（本课核心）

#### 旋律创作五要点

| 要点 | 内容 | 程度 |
|------|------|------|
| **① 音域** | 高声部控制在**小字一组 C ~ 小字三组 C**（两个八度） | 参考，非死理 |
| **② 最高音后置** | 旋律最高音放在乐段**后半部分**（如 8 小节乐段放第 5-7 小节） | 黄金分割点原理 |
| **③ 高低反向** | 旋律与低音**尽量**反向进行 | 大概 3:1 即可 |
| **④ 平稳为主** | 平稳进行（≤三度）与跳进（>三度）比例约 **3:1** | 参考比例 |
| **⑤ 跳进后反向** | 跳进后**尽量**反向进行 | 大概 3:1 即可 |

#### 要点详解

**① 音域**：过高刺耳、过低浑浊。C4~C6 是女高音和大多数旋律乐器的舒适区。

**② 最高音后置（黄金分割点 0.618）**：
- 高潮需要前面的平淡做对比铺垫，所以放在后半段
- 8 小节的乐段：最高音放在第 5/6/7 小节
- 第一乐句要**平淡**，为第二乐句的爆发留空间
- 第一乐句的跳进也要节制（4-6 度即可，7-8 度留给后半段）

**③④⑤**：这三条都不是死规则，「尽量」≈ 大概 3:1 的比例即可。

### 第三步实操演示（西蒙手把手）

以 C 大调 8 小节平行乐段为例：

**第一乐句（前 4 小节）**：
1. 选起始音：结合音域 + 最高音后置 + 高低反向 → 排除过高/过低的选项
2. 逐音往下写：优先平稳 + 反向
3. 连续 2-3 次平稳后 → 做一次大于三度的跳进
4. 跳进后优先反向 + 平稳
5. 半终止处：可选 D 或 K₄₆→D，都试试，哪种好听用哪种

**第二乐句（后 4 小节）**：
1. **开头照抄第一乐句的前 2-3 个音**（平行乐段的「同头」）
2. 在同样的位置做跳进，但**幅度更大**（如第一句跳 4 度，第二句跳 8 度）
3. 这就是「控制变量法」——同样位置/同样动作，只改变幅度，对比感更强
4. 跳进到达最高音后 → 反向平稳下行 → 收尾

**最后一步**：填两个内声部（同第 14 课的方法，检查五八度等）

## 三、关键心法

> **旋律没有唯一正确答案。** 条例只是没头绪时的参考，你觉得哪种好听就用哪种。当灵感来了，相信自己的感觉，不必纠结是否完全符合条例。

### 低音题 vs 高音题 对比

| | 高音题（已有） | 低音题（新增） |
|------|-----------|-----------|
| 给定 | 旋律 | 低音 |
| 自定 | 和弦选择 + 低音 + 内声部 | 旋律 + 内声部 |
| 训练目的 | 和弦选择能力 | 旋律创作能力 |
| 自由度 | 低 | 高 |

> 两种题都要持续练习。以后拿到一条旋律/低音，先分析乐句乐段结构和终止式布局，再开始写。

## 四、本课总结

| 要点 | 一句话 |
|------|--------|
| 低音题本质 | 低音已锁定和弦，在此框架内创作旋律 |
| 旋律音域 | 高声部 C4~C6 最合适 |
| 最高音策略 | 放乐段后半段（黄金分割），第一句平淡留空间 |
| 平稳 vs 跳进 | 约 3:1，平稳占主导 |
| 高低反向 | 尽量反向，约 3:1 |
| 平行乐段 | 第二句开头照抄第一句，在同样位置做更大跳进 |
| 核心态度 | 没有唯一解，好听最重要；条例只是参考 |

### 低音题速查流程

```
拿到低音 → ① 判断调性（调号 + 最后一个低音）
          → ② 标注和弦 + 分析乐句结构/终止式
          → ③ 创作旋律（五要点）
          → ④ 填内声部（检查五八度）
```
