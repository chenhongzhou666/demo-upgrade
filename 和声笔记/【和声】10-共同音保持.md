# 【和声】第10课 — 共同音保持

> 讲师：西蒙（三分钟音乐社）
> 来源：B站 BV1mv411F7aY
> 提取时间：2026-07-14

## 本课定位

第9课讲了同和弦转换（同和弦内部可以跳），本课讲共同音保持（不同和弦间相同音可以合并）。这两个技巧会**贯穿整个和声学习**，是化解「配不了」难题的两大利器。

---

## 一、什么是共同音保持

如果某个声部在连续和弦中的音**相同**，可以把它们合并成一个长音符（用延音线连接），只给第一个音头。

```
改写前：每一拍四个声部都有新音头
改写后：没变的声部保持长音，只有改变的声部给新音头
```

### 为什么要做？

| 问题 | 不改写 | 改写后 |
|------|--------|--------|
| 声部独立性 | 每次四个音头同时响，大家互相抢注意力 | 改变中的声部突出，不变的声部退后 |
| 层次感 | 缺少重点与非重点的对比 | 注意力交替转移，音乐更鲜活 |
| 死板感 | 所有声部永远同样节奏，呆板 | 有呼吸感、生命力 |

> **核心心法**：突出正在改变的声部，削弱没有在改变的声部。让「变化」成为听众注意力的焦点。

---

## 二、不是必须，是审美选择

共同音保持**没有绝对的对错**，两种写法都正确。这是审美层面的选择，不是规则层面。

### 新手建议：按强拍/弱拍来

| 拍位 | 建议 | 原因 |
|------|------|------|
| **强拍 / 次强拍** | 尽量多出现音头（至少2-3个） | 符合「强」的感觉 |
| **弱拍** | 尽量做共同音保持，少出现音头 | 符合「弱」的感觉 |

强拍只有一个音头甚至没有 → 听感上失去了「强拍」的感觉，大众会觉得违和。

---

## 三、延音线的写法规范

四部和声单行谱表记两个声部时，延音线朝向与普通单声部**相反**：

| 声部 | 延音线朝向 |
|------|-----------|
| 高声部、次中声部 | 向上凸 ↑（凸向符杠一侧） |
| 中声部、低声部 | 向下凸 ↓（凸向符杠一侧） |

> 这样写是为了美观，避免延音线互相交叉看不清。

## 四、逆向思维①：长旋律音配多个和弦

### 核心洞察

旋律本身的长音符（二分音符、全音符等），可以当作是**高声部的共同音保持**，从而在下方配**多个不同的和弦**。

### 例子（C大调 全音符 do-si-sol-la）

**之前做法**（一个音一个和弦）：
```
do=T → si=D → sol=T → la=S
= T→D→T→S　　略显呆板
```

**利用共同音保持**（给长音换和弦）：
```
do 四拍：前两拍T + 后两拍S
si 两拍：D（只一个选择，目前）
sol 四拍：前两拍D + 后两拍T
la 四拍：S（只一个选择，目前）
= T→S→D→D→T→S　　和声丰富了！
```

同时化解了原来 sol→la 的 D→S 逆功能进行（在 sol 内先接到 T，再接 la 的 S，变成了 T→S 的正向进行）。

### 适用条件

- 适用于**时值较长**的音符（四分音符及以上）
- 八分/十六分音符如果还换和弦 → 太混乱，听不清
- 以后和弦学多了，每个长音都有更多换和弦的选择

### 经典例子：肖邦 E小调前奏曲

旋律大量使用长音符，左手靠和弦变化推动音乐。这就是「一个旋律音配多个和弦」的典范。

---

## 五、逆向思维②：结合同和弦转换调整排列

### 综合运用

当遇到「现有排列接不上下一个和弦」（比如会出现声部超越）时：

```
策略：
1. 高声部长音 → 当作共同音保持（旋律不动）
2. 下方三声部 → 做同和弦转换（可以跳！）
3. 调整完排列后 → 顺利接下一个和弦
```

### 例子（G大调 T-S-T-D）

前三个和弦 T-S-T 有且只有一种正确排列，但这种排列下第四个 D 和弦会出现声部超越。

**解法**：在第三个 T（长音）内，利用同和弦转换把中声部降下来 → 再接到 D 时就没有声部超越了。

---

## 六、两大利器对比

| | 同和弦转换（第9课） | 共同音保持（第10课） |
|--|-------------------|---------------------|
| 场景 | 连续同和弦 | 连续不同和弦但某声部音相同 |
| 特权 | 声部可跳超过三度 | 合并成长音符，不重复发音 |
| 组合用法 | 在高声部保持时，下方做同和弦转换来调整排列 |
| 共同目标 | 化解配不上的困境，让声部连接更顺畅 |

---

## 七、总结

| 要点 | 一句话 |
|------|--------|
| 定义 | 相同音不重复发音，合并成长音符 |
| 作用 | 突出变化中的声部，增加层次感 |
| 不是必须 | 审美选择，非规则约束 |
| 新手建议 | 强拍多音头，弱拍多保持 |
| 逆向用法① | 长旋律音 = 高声部保持 → 下方换和弦 → 丰富和声 |
| 逆向用法② | 高声部保持 + 下方同和弦转换 → 调整排列 → 接好后面 |
