## 本课定位

🎯 和上节课（经过的四六和弦）是**姊妹篇**。两个概念来自同一个乐理基础——第17课讲过的经过音与辅助音的区别。

---

## 一、从乐理到和声：经过 vs 辅助

你在乐理第93课和本课程第17课都学过：

| | 经过音 | 辅助音 |
|------|------|------|
| **特征** | 同向连续级进，左右音**不同** | 左右音**相同**，中间音级进插入 |
| **例子** | do-**re**-mi | do-**re**-do |
| **本质** | 连接不同音，有推进感 | 同一音的装饰，稳定中的点缀 |

上升到和弦级别，完全同理：

| | 经过的四六和弦（第20课） | 辅助的四六和弦（本课） |
|------|------|------|
| **左右和弦** | **不同**（T vs T₆，或 S vs S₆） | **相同**（T vs T，或 D vs D） |
| **低音** | **级进**（do-re-mi 或 fa-sol-la） | **保持**（do-do-do 或 sol-sol-sol） |
| **听感** | 有推进感、运动感 | 稳定中做点缀，不动的基调 |

## 二、组合一：T → S₄₆ → T

### 构造（C大调示例）

```
T:    低音 do  上方: mi-sol-do（根音+三音+五音各一）
S₄₆:  低音 do  上方: fa-la-do（根音+三音+五音各一）→ 重复五音！
T:    低音 do  上方: mi-sol-do
```

### 声部进行（最平稳形式）

| 声部 | T → S₄₆ → T | 进行 |
|------|:---:|:---:|
| 高声部 | do → do → do | 共同音保持 |
| 中声部 | mi → fa → mi | 小二度往返 |
| 次中声部 | sol → la → sol | 大二度往返 |
| 低声部 | do → do → do | 共同音保持 |

> **低音全程不变** → 主功能基调贯穿始终，S₄₆ 只是中间轻轻的装饰。

### 六个要点（①②④⑤与经过类完全一样）

| 要点 | 内容 |
|------|------|
| **①** 不强于两侧 | S₄₆ 在弱拍/弱位（辅助过渡性质） |
| **②** 两侧相同 + 低音保持 | 左右都是 T，低音 do-do-do（这是与经过类的核心区别） |
| **③** 重复五音 | S₄₆ = 五音在低音 + 上方根三五音各一 |
| **④** 正反皆同 | T→S₄₆→T 完全左右对称，正反一样 |
| **⑤** 上方轮转 | 上方三声部（do-do-do、mi-fa-mi、sol-la-sol）谁做高声部都行 |
| **⑥** 可做三音跳进 | 高声部/次中声部可跳进：sol-la-sol → mi-la-sol（此时 mi-fa-mi → sol-fa-mi） |

### 识别该用的场景

旋律出现以下形态时可考虑：

| 旋律形态 | 级数 | 配法 |
|---------|:---:|------|
| 同音保持模式 | 1-1-1 级 | T→S₄₆→T（下方改变和声） |
| mi-fa-mi | 3-4-3 级（或 7-1-7 级） | T→S₄₆→T |
| sol-la-sol | 5-6-5 级 | T→S₄₆→T |

## 三、组合二：D → T₄₆ → D

### 构造（C大调示例）

```
D:    低音 sol  上方: re-ti-sol
T₄₆:  低音 sol  上方: do-mi-sol → 重复五音！
D:    低音 sol  上方: re-ti-sol
```

### 声部进行（最平稳形式）

| 声部 | D → T₄₆ → D | 进行 |
|------|:---:|:---:|
| 高声部 | re → re → re | 共同音保持 |
| 中声部 | ti → do → ti | 小二度往返 |
| 次中声部 | sol → sol → sol | 共同音保持 |
| 低声部 | sol → sol → sol | 共同音保持 |

> 属功能基调贯穿始终，T₄₆ 只是轻轻的经过装饰。

### 第六点变体（三音跳进）

| 常规声部 | 三音跳进变体 |
|---------|------------|
| re-mi-re | ti-mi-re |
| ti-do-ti | re-do-ti |

## 四、四组经过/辅助和弦速查总表

### 经过类（左右和弦不同，低音级进）

| 组合 | 低音 | 上方三声部（可轮转） |
|------|:---:|------|
| T→D₄₆→T₆（或反向） | 1→2→3 / 3→2→1 | mi-re-do / do-do-do / do-ti-do |
| S→T₄₆→S₆（或反向） | 4→5→6 / 6→5→4 | la-sol-fa / do-do-do / fa-mi-fa |

### 辅助类（左右和弦相同，低音保持）

| 组合 | 低音 | 上方三声部（可轮转） |
|------|:---:|------|
| T→S₄₆→T | 1→1→1 | sol-la-sol / mi-fa-mi / do-do-do |
| D→T₄₆→D | 5→5→5 | re-mi-re / ti-do-ti / sol-sol-sol |

### 通用规则

- ⚠️ 数字均为**级数**（首调），不是固定调唱名
- 上方三声部可任意轮转（谁做高声部都行）
- 中间的 S₄₆/D₄₆/T₄₆ 在弱拍/弱位、重复五音
- 经过类有推进感，辅助类是稳定中点缀

## 五、新用法：S₄₆ 做变格补充终止

更新第19课的知识——变格补充终止现在有三种选择：

| 方式 | 听感 | 原因 |
|------|------|------|
| S → T | 终止感较明显 | 低音 fa→do 变化 |
| S₆ → T | 同上 | 低音 la→do 变化 |
| **S₄₆ → T（新）** | **最稳定、最柔和** | 低音 do→do 保持不变 |

> 用 S₄₆ 做变格补充终止时，低音一直停在主音上不动，听上去是「主功能基调下的轻轻点缀」，比其他两种更含蓄。

不是谁更好——根据不同音乐需要选择。

---

## 六、实战提示

拿到旋律配和弦前，除了标出终止式/K₄₆/变格补充终止外，也对照速查表先标出**可以做经过/辅助四六和弦的地方**：

- 级进三音组（1-2-3、3-2-1、4-5-6、6-5-4 等）→ 经过类
- 同音保持或往返（1-1-1、5-5-5、3-4-3、5-6-5 等）→ 辅助类

难点不在于配法（配法都很简单），在于**能不能想到可以这么配**。

## 七、本课总结

| 要点 | 经过（第20课） | 辅助（本课） |
|------|------|------|
| 乐理原型 | 经过音（do-re-mi） | 辅助音（do-re-do） |
| 两侧和弦 | 不同 | **相同** |
| 低音 | 级进 | **保持** |
| 听感 | 推进、运动 | 稳定、点缀 |
| 共有组合 | T↔D₄₆↔T₆、S↔T₄₆↔S₆ | T→S₄₆→T、D→T₄₆→D |
| 特殊点 | 六和弦可重复五音 | 可做三音跳进 |
| 共同规则 | 弱位、重复五音、上方轮转、正反皆可 | 同左 |
