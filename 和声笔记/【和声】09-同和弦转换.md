# 【和声】第9课 — 同和弦转换

## 本课定位

第7课留下的悬念——「同和弦转换时可以跳超过三度」——本课彻底讲清楚。这个概念很简单但很实用。

---

## 一、定义

> **同和弦转换**：连续多个音配的都是**同一个和弦**，但纵向上的排列不完全一样。

```
例：C大调  do → mi → sol
        T     T     T     ← 三个都是T，但排列不同 = 同和弦转换
```

---

## 二、核心规则：可以跳超过三度

### 第5课的规则需要加限制条件

| 情况 | 声部进行限制 | 原因 |
|------|------------|------|
| **不同和弦连接**（T→S、S→D 等） | ≤ 三度，尽量平稳 | 和声效果在改变，重点在此，声部需平稳衬托 |
| **同和弦转换**（T→T、S→S 等） | **可以超过三度**（四度、五度、六度都行） | 和声效果没改变，重点不在和声变化上，无需限制 |

### 为什么？

核心思路是「**和声效果改变**」与「**声部进行**」的平衡：

- **不同和弦连接时**：和声整体听感/情绪在改变 → 这是重点 → 声部要平稳来衬托这个改变
- **同和弦转换时**：和声整体听感没改变 → 没必要强行限制声部在三度内 → 可以自由跳动

### ⚠️ 其他规则照常生效！

同和弦转换只解除了「≤ 三度」的限制，以下规则**依然必须遵守**：

- ❌ 声部交叉/超越
- ❌ 平行/反向/隐伏五八度
- ❌ 四部同向
- ✅ 重复根音
- ✅ 上方三声部 ≤ 纯八度

---

## 三、同和弦转换的两大作用

### 作用一：自由调整排列，为后面铺路

当后续和弦需要声部在较高/较低位置才能正确连接时，利用同和弦转换提前把声部调到需要的位置。

**例子（C大调 do-mi-si → T-T-D）**：

```
错误思路：mi保持三度内平稳 → 内声部被压得太低
         → D和弦上方声部超过纯八度 ❌

正确思路：mi的同和弦转换中，内声部跳四度往上提
         → D和弦轻松接上，一切正确 ✅
```

### 作用二：处理旋律的大跳进

当旋律出现五度以上的大跳时，用同和弦转换是最自然的解法。

**例子（C大调 mi-do，六度下跳）**：

- mi 只能配 T
- do 如果配 S：无论怎么排列都必然出错（声部超越 / 上方超八度 / 没重复根音）
- do 配 T：同和弦转换，轻松搞定 ✅

> 看到大跳进心里不用慌——大概率就是同和弦转换。

---

## 四、更新后的「平稳」规则

| 连接类型 | 声部进行 |
|----------|---------|
| 不同和弦连接 | 内声部 ≤ 三度 |
| **同和弦转换** | **可以跳进（四度、五度、六度等），完全正常** |

---

## 五、总结

| 要点 | 一句话 |
|------|--------|
| 定义 | 连续同和弦但排列不同 |
| 核心特权 | 声部可以跳超过三度 |
| 原因 | 和声没变 → 不需要平稳来衬托变化 |
| 作用一 | 自由调排列，为后面和弦铺路 |
| 作用二 | 处理旋律大跳进 |
| 底线 | 其他6条规则照常遵守 |
