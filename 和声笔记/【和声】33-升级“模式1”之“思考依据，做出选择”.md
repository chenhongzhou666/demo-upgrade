## 二、核心方法论：主动找依据

### 为什么需要依据

写出 7-8 种版本后如果不加筛选 → 选择困难。有依据 = 有信心的判断。

### 思考流程

```
写出多版本 → 反复对比听 → 判断哪个舒服/不舒服
→ 猜测原因 → 验证猜测 → 提炼为依据 → 用依据缩小范围
```

### 例子：四种节奏版本逐个听，提炼出以下依据。

## 三、节奏选择的四个依据

### 依据一：复拍子应覆盖次强拍

| 拍号 | 类型 | 强弱规律 | 建议 |
|------|------|------|------|
| **四四拍** | 复拍子(2+2) | 强-弱-次强-弱 | 次强拍（第3拍）应重新奏和弦 |
| **八六拍** | 复拍子(3+3) | 强-弱-弱-次强-弱-弱 | 次强拍（第4拍）应重新奏和弦 |

> 全音符覆盖整个小节 → 次强拍无声 → 断断续续跟不上律动。

### 依据二：结尾和弦应在强拍/次强拍

和弦**最后一次出声**应落在强拍或次强拍，否则终止感弱，有「戛然而止」的错愕感。

**原因**：强拍上的和弦才能被强调与巩固，带来足够的终止感/结束感。

### 依据三：节奏密度 ≤ 主旋律节奏密度（柱式）

**柱式和弦**的节奏密度尽量**不超过**主旋律的平均节奏密度，否则伴奏抢戏。

| 计算方法 | 
|------|
| 主旋律节奏密度 = 总音符数 ÷ 总拍数（拍/音） |
| 例：40个音符 ÷ 32拍 = 1.25 音/拍 |

- 主旋律 1.25 音/拍 → 柱式用四分音符（1次/拍）合适（1 < 1.25）
- 八分音符（2次/拍）> 1.25 → 抢戏，排排除
- 16分音符 → 更抢戏

### 依据四：分解织体的节奏密度可放宽到 2 倍

分解织体适用不同标准，有两个原因：

| 原因 | 解释 |
|------|------|
| **需更多拍数** | 分解需逐音弹完和弦（如CEG三个音需2拍），柱式瞬间弹出 |
| **每次信息量小** | 分解每次只发一个音 ≠ 柱式一次3-4个音，抢戏感弱 |

> 主旋律 1.25 音/拍 → 分解可用八分音符（2次/拍），2 ≤ 1.25×2=2.5 ✅

---

## 四、分解织体的关键补充

### 低音重启 = 和弦感知的起点

在分解织体中，**每次低音重新出现**都会被听觉感知为「和弦新一轮开始了」。

> 必须让低音的每次重新出现落在**强拍、次强拍或弱拍强位**。

否则会产生「1.5+1.5+1」的割裂感，违背四四拍「2+2」的强弱规律。

**修改方法**：让每次低音重启与强拍/次强拍对齐 → 音符不够时**向高处补充最近的和弦构成音**（如写完 C-E-G 后再补一个高八度 C）。

### 结尾处理

即使全曲用分解织体，**最后一个结尾和弦应改用柱式**——单独一个分解音做结尾太单薄，柱式和弦才有饱满的终止感。

---

## 五、依据的本质定位

> **依据 ≠ 死规则，依据 = 辅助判断的工具。**

- 有明确主观喜好时：跟自己的耳朵走
- 没有明确喜好/犹豫时：参考依据来做决定
- 依据让人更有**信心**地选择，而不是强制约束

## 八、节奏选择速查

| 场景 | 怎么做 |
|------|------|
| 四四拍/八六拍 | 次强拍上重新奏一次和弦 |
| 结尾小节 | 让最后一次和弦出现在强拍或次强拍 |
| 柱式选节奏 | 和弦密度 ≤ 旋律密度（拍均音符数） |
| 分解选节奏 | 和弦密度 ≤ 旋律密度×2 |
| 分解织体低音 | 每次低音重启对齐强拍/次强拍 |
| 分解结尾 | 最后改柱式和弦以增加饱满终止感 |
