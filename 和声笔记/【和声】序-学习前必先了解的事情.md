# 【和声】序 — 学习前必先了解的事情

> 来源：B站 BV1j14y167yt
> 提取时间：2026-07-13

## 一、和声课学什么

**核心目标：给旋律配和弦的能力**

具体包含三个层面，互相促进：

| 能力 | 说明 |
|------|------|
| 给旋律配和弦 | 一段旋律用什么和弦、是否需要转位、内部怎么连接 |
| 提升旋律创作 | 通过和声布局写出更有逻辑、更有出彩点的旋律，而非纯靠感觉 |
| 分析他人作品 | 看懂、听懂别人作品里的创作思路，化为己用 |

## 二、前提条件

**基础乐理全部内容必须掌握**，尤其是：
- 五线谱（全程用五线谱讲解，简谱无法讲全面）
- 音程
- 和弦
- 调式

> 对标「三分钟音乐社」乐理第二季（不含加星号课程）

## 三、课程路线与时间

- **科班路线**：功能体系 → 四部和声训练 → 流行风格编配
- **学习时间**：至少 1-2 年（科班生大学里也至少学一年）
- **不能速成**：两三个月搞懂不现实

## 四、怎么学怎么练

### 必备工具
1. **键盘类乐器**（首选钢琴/电钢/电子琴/MIDI键盘）—— 配完和弦必须弹出来听
2. **打谱软件**（Sibelius / LilyPond / MuseScore / Overture）—— 弹不了就输入软件播放
3. **宿主软件**（Cubase / Logic / FL Studio）—— 编曲相关必备

### 核心方法
- **练习 → 反馈 → 批改 → 循环**，这是最重要环节
- 和声不同于基础乐理（客观知识，学会记住就好），它是主观创作，需要大量练习
- 建议找一对一老师，节约时间少走弯路

## 五、关键认知

- 某一个教程无法涵盖所有时代/风格/体系的和声
- 系统学过一门和声课后，再去研究其他风格/体系会好理解得多
- 学好功能体系后已能满足绝大部分需求
- 只看教程没有反馈批改 = 自己买本书看，效果大打折扣
