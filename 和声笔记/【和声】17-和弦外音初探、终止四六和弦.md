## 第一部分：和弦外音初探

### 一、和弦内音 vs 和弦外音

| 概念 | 定义 |
|------|------|
| **和弦内音** | 属于该和弦构成音的音（目前阶段 = 根音、三音、五音） |
| **和弦外音** | 出现在该和弦中但不属于构成音的音 |

### ⚠️ 重要认知

**和弦内音/外音的界定没有统一标准**，取决于：
- 音乐时期（古典 vs 现代）
- 音乐风格（传统三和弦 vs 九和弦/十一和弦）
- 学习阶段（目前只学了三和弦，所以根三五以外都是外音；以后学了属七/属九后逐渐纳入）

> 不要纠结某个音「到底算内音还是外音」，重要的是理解这个音在和弦里起的**作用**和带来的**听觉效果**。

### 二、和弦外音的五种形式

#### 前三种 = 弱和弦外音（弱外音）

共同关键词：**不强于** —— 总是处于相对较弱的位置

| 类型 | 定义 | 判断要点 |
|------|------|---------|
| **① 经过音** | 同方向连续级进，处于中间的那个音 | 不强于两侧音；可单独配和弦（= 不是外音），也可延续前方和弦（= 是外音） |
| **② 辅助音** | 两侧音相同，中间音级进插入 | 同经过音；两侧音一样 vs 经过音两侧不同 |
| **③ 先现音** | 后一和弦的某个音提前在前一和弦出现 | 不强于后方真正要预示的音；可能/可能不是外音 |

**重要修正**（比乐理课更严谨）：不强于 ≠ 必须处于弱拍弱位。只要**不强于两侧音**即可（可以是同强、同为弱拍等）。

> 目前练习中仍要求给经过音/辅助音单独配和弦，暂不做成和弦外音形式。先练好「单独配」的能力，以后再选择。

#### 后两种 = 强和弦外音（强外音）

共同关键词：**不弱于** —— 总是处于相对较强的位置，需要**在同一和弦背景下解决到内音**

| 类型 | 定义 | 区别 |
|------|------|------|
| **④ 延留音** | 前一和弦某声部的音延续到后一和弦，成为和弦外音 | 来源 = 前一和弦保持来的 |
| **⑤ 倚音** | 直接出现在某和弦中，既不是延留也不是构成音 | 来源 ≠ 前一和弦保持来的（和声课的「倚音」≠ 乐理课装饰音「倚音」） |

> 注意：延留音/倚音「可能」是和弦外音也可能不是，取决于所处位置和弦的构成音中是否包含它。我们讨论的主要是它们「是」和弦外音的情况。

### 三、强外音的三个要点

| 要点 | 内容 |
|------|------|
| **① 位置** | 处于强拍/强位（或至少不弱于后面的解决音） |
| **② 解决方向** | 大多**向下级进解决**到内音（二度音→根音、四度音→三音、六度音→五音；只有七度音向上解决到根音） |
| **③ 作用** | **延迟解决**——把本可一步完成的解决拆成两步，增加层次感和期待感 |

#### 解决方向速查（同一和弦背景下，外音→内音）

| 外音类型 | 解决到 | 例（C大调T背景） |
|---------|--------|-----------------|
| 二度音 | ↓ 根音（优先）/ ↑ 三音（备选） | re → do |
| 四度音 | ↓ 三音 | fa → mi |
| 六度音 | ↓ 五音 | la → sol |
| 七度音 | ↑ 根音 | si → do |

> ⚠️ 这是**同一和弦内**外音→内音的解决，不是两个不同和弦之间的解决（如属七→主的七一三音解决是另一回事）。

### 四、延留音的两种形式

| 形式 | 特点 | 紧张感 |
|------|------|--------|
| **延音线延续** | 延留时不重新发新音头 | 较弱 |
| **重新发出音头** | 延留处重新击键 | 较强（重新强调不协和音程） |

> 选哪种取决于听觉需要。

## 第二部分：终止四六和弦（K₄₆）

### 一、什么是 K₄₆

**定义**：用在半终止或全终止处的，涉及了四度音与六度音这两个强外音的**属功能和弦**。

标记：**K₄₆**（K = 终止，来自德语/俄语；⁴₆ 表示根音上方的四度音和六度音）

### 二、K₄₆ 的构造

| 声部 | 音 | 说明 |
|------|-----|------|
| 低音 | **属音**（根音，重复） | 最核心，加强属功能性质 |
| 上方声部 | 根音（重复）+ 四度音 + 六度音 | 共 4 个声部，两个根音 |

例 C大调 K₄₆ = { 低音 G，上方 G(重复根音) + C(四度音) + E(六度音) }

### ⚠️ 重要：K₄₆ 不是主和弦第二转位！

外观上 K₄₆ 的构成音（G-C-E）看起来像 C 大三和弦的第二转位，但：

| 误解 | 真相 |
|------|------|
| 看起来像主和弦 G-C-E | 实际是**属功能和弦**：根音G + 外音C(四度) + 外音E(六度) |
| 功能：主 | 功能：**属**（不稳定，紧张，需解决） |

> 用和弦外音的概念理解 K₄₆ 才是最准确的：它是属功能和弦的根音 + 两个强外音。

### 三、K₄₆ 使用的三个关键条件

#### 条件一：只能用在终止处

只能用在**半终止**或**全终止**处。用在其他地方会让不应该断句的地方产生断句感。

#### 条件二：后面必须接原位属和弦（K₄₆→D 绑定）

- K₄₆ 的四度音 ↓ 解决到 D 的三音（级进下行）
- K₄₆ 的六度音 ↓ 解决到 D 的五音（级进下行）
- 95%+ 的情况后面接**原位**属和弦（属三或属七）

> K₄₆ + D = 不可分割的「属功能组合」，视为一个大号属功能

#### 条件三：K₄₆ 不弱于 D

| 维度 | 要求 |
|------|------|
| **节拍强度** | K₄₆ ≥ D（不能 K₄₆ 在弱拍、D 在强拍） |
| **时值长度** | K₄₆ ≥ D（不能 K₄₆ 比 D 短） |

违反任一条 → 运行不畅，宁愿不配 K₄₆。

### 四、终止式优先级更新

| 位置 | 新的优先级 |
|------|-----------|
| **半终止** | ① **K₄₆ → D**（正格半终止） > ② 单独的 D > ③ S 或不完满 T |
| **全终止** | ① **S → K₄₆ → D → T**（完全终止，最优） > ② S → D → T > ③ K₄₆ → D → T（无 S） > ④ D → T > ⑤ S → T |

### 五、K₄₆ 的前方

| 场景 | K₄₆ 前可用 | 不可用 |
|------|-----------|--------|
| **半终止** | T 或 S（两者频率差不多） | D / D₆（属功能→属功能会形成和声切分） |
| **全终止** | 优先 S（形成完全终止）；也可 T | D / D₆（同理） |

### 六、K₄₆ 可长可短

K₄₆ 可以持续多个音，在内部做**同和弦转换**。拿到旋律时要观察是否有连续多个音都是 K₄₆ 的构成音。

### 七、全终止 K₄₆→D 的特殊规则 ⭐

| 场景 | 规则 |
|------|------|
| **半终止** K₄₆→D | 旋律**必须**做外音到内音的级进解决（不能跳进） |
| **全终止** K₄₆→D | 旋律**可以跳进**（不做内部解决，把不稳定感积攒到 D→T 一起释放） |

**原因**：全终止最后的 D→T 是功能间外部矛盾的解决，优先级更高。当内部矛盾与外部矛盾冲突时，优先处理外部矛盾。

  - K₄₆→D 温柔级进解决 = 一步一步下台阶（温柔）
  - K₄₆→D 跳进 = 把紧张积攒到 D→T 一次性释放（畅快）

> 全终止 K₄₆→D 做跳进时，**在无可避免时可以允许声部超越和四部同向**。这是本课程第一条「可以违反规则」的规则。

**为什么低音往下跳八度更好？**
  - 更低频 = 更多不稳定感（人体共振频率效应）
  - D 根音↑四度到 T 根音比 ↓五度的解决感更强

### 八、K₄₆ 使用条件速查

| 条件 | 要求 |
|------|------|
| 位置 | 仅半终止/全终止 |
| 后接 | 必须接原位 D（K₄₆→D 绑定） |
| 节拍 | K₄₆ ≥ D 的节拍强度 |
| 时值 | K₄₆ ≥ D 的时值长度 |
| 半终止旋律 | 不能跳进（必须做外音到内音解决） |
| 全终止旋律 | 可跳进可不跳，跳进时允许声部超越/四部同向 |

## 三、本课总结

### 和弦外音五类型

```
             ┌─ 弱外音（不强于后方音）
             │   ├─ ① 经过音：同向连续级进的中间音
             │   ├─ ② 辅助音：两侧音相同的中间级进音
             │   └─ ③ 先现音：提前预示后和弦的音
和弦外音 ───┤
             └─ 强外音（不弱于后方解决音，需在同一和弦内解决）
                 ├─ ④ 延留音：前一和弦保持来的
                 └─ ⑤ 倚音：非延留、非构成的强外音
```

### K₄₆ 核心要点

| 要点 | 一句话 |
|------|--------|
| 本质 | 属功能和弦的根音 + 四度音 + 六度音（两个强外音） |
| 不是 | 主和弦第二转位（只是外观像） |
| 绑定 | K₄₆ 后面必须接原位 D |
| 位置 | 仅用于终止处，且 K₄₆ 不弱于 D |
| 半终止 | 旋律不能跳进（必须级进解决） |
| 全终止 | 旋律可以跳进（允许声部超越/四部同向） |

### 拿到旋律的实战流程（更新版）

1. **判断调性**
2. **分析结构**：划分乐句，确定半终止/全终止位置
3. **布局终止式**：优先看能不能用 K₄₆→D（或 S→K₄₆→D→T）
4. **从头配和声**：两个外声部优先，再填内声部
