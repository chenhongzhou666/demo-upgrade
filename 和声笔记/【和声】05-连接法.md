## 本课定位

前两课讲了**纵向**（单个和弦怎么排列），本课讲**横向**（和弦与和弦之间怎么连接）。

---

## 一、两种连接法（名词概念）

| 连接法 | 定义 | 判定 |
|--------|------|------|
| **旋律连接法** | 所有声部的音**都**发生改变 | 四个声部全变了 |
| **和声连接法** | **至少有一个**声部的音没变 | 有共同音保留 |

> ⚠️ 这两种连接法**没有优劣之分**，实操中根本不会去想「该用旋律连接法还是和声连接法」。这只是一个描述性词汇，和「密集排列」「开放排列」一样，仅用于方便讲解。

---

## 二、核心实操原则：背景声部要尽量平稳

### 前景 vs 背景

```
高声部(S) ──── 前景（主角）── 起伏要大，吸引注意力
中声部(A) ─┐
            ├── 背景（配角）── 要平稳，不抢戏
次中声部(T)─┘
低声部(B) ──── 男2号 ─────── 介于主角与配角之间
```

**角色分工逻辑**：
- **高声部**：人耳最敏感，天然适合当主角 → 需要起伏来制造听觉刺激
- **内声部（A+T）**：夹在中间，最纯粹的背景 → 必须平稳，否则抢戏
- **低声部**：人耳第二敏感 + 决定和弦性质（低音是什么决定和弦是什么）→ 不一定能做到平稳，有时根本没得选

### 抢戏的后果

如果背景声部经常做大跳进，听众的注意力会被强行扯过去 → 分不清谁是主角 → 听觉混乱。

### 平稳的标准

> **三度及三度以内** 的音变化算平稳

为什么不是二度？因为很多和弦之间，即便最平稳的连接也会出现三度进行。例如 C大调 T→S：

```
T (CEG重复C)  →  S (FAC重复F)
中声部: C → A  (三度)
次中声部: E → C (三度)
```

这已经是最平稳的方式了，做不到全部二度以内。所以标准定为三度。

## 三、实操注意事项

### 目前你只需要记住三个点

| # | 要点 | 来源 | 维度 |
|---|------|------|------|
| 1 | 上方三声部两两 ≤ 纯八度 | 第4课 | 纵向 |
| 2 | 重复根音 | 第4课 | 纵向 |
| 3 | **内声部尽量平稳**（≤ 三度） | 第5课 | 横向 |

> 实操时**根本不需要想**「密集/开放排列」「旋律/和声连接法」「根音/三音/五音旋律位置」这些词，只围绕上面三条规则做就行。

### S → D 的低音规则

S → D 的低音**只能写上行二度**，不能写下行七度。例如 C大调：F → G（上行二度），不要 F → 低八度的 G（下行七度）。

（下行七度为什么不行？下节课讲）

### 连接法的实操步骤示例

以 D大调 T → S → D → T 为例，已知高声部和低声部，填充内声部：

```
步骤：
1. 推算目标和弦的构成音（查参考表）
2. 确定需要哪些音（低音+高声部已占2个，剩下2个分配给内声部）
3. 内声部的音如何排列 → 试！找最平稳的接法（前后声部变化最小）
4. 检查：上方三声部是否 ≤ 纯八度？是否重复根音？
```

**核心心法**：不用想那些名词，就抓一个点——**内声部怎么接最平稳就怎么来**。

---

## 四、总结

| 概念 | 一句话 |
|------|--------|
| 旋律连接法 | 所有声部都变了 |
| 和声连接法 | 有共同音保留 |
| 前景 | 高声部（主角），需要起伏 |
| 背景 | 内声部（配角），必须平稳 |
| 平稳标准 | 音的变化 ≤ 三度 |
| S→D 低音 | 只能上行二度 |
| 实操心法 | 围绕三条规则，追求内声部最平稳 |
