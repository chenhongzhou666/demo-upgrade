## 本课定位

🎯 调内七个和弦的最后一个——III 级（DTⅢ）。内容很短，用法也窄。至此调内七个和弦全部学完。

---

## 一、为什么只在大调用

### 小调的三级和弦

以 a 和声小调为例：**do-mi-sol♯ = 增三和弦** → 极度不协和 → 几乎不用。

```
a自然小调：do-mi-sol = 大三和弦 → 可用
但自然小调还没学，暂放
```

→ **本课只讲大调的 III 级。**

---

## 二、结构与标记

### 构成

```
C大调 III 级：E-G-B（小三和弦，协和）

E-G ← 主功能 T 的音（mi-sol 也是 T 的构成音）
G-B ← 属功能 D 的音（sol-si 也是 D 的构成音）
```

→ 复合功能和弦

### 功能偏向来决定标记顺序

| 和弦 | 包含的功能音 | 偏向 | 标记 |
|------|------|:---:|------|
| VI 级 (A-C-E) | C-E = T，A-C = S | 偏 **T** | **TS**Ⅵ |
| **III 级** (E-G-B) | E-G = T，G-B = D | 偏 **D** | **DT**Ⅲ |

> 有属音 G 但没有主音 C → 偏向属功能 → D 写前面。

---

## 三、使用限制

| 限制 | 原因 |
|------|------|
| **只用原位三和弦** | 功能性不明确，转位更模糊（同 TSⅥ） |
| **只用原位，不用七和弦** | 古典时期主要用三和弦形式（七和弦后面再补） |
| **重复根音** | 没有特殊理由去重复三音/五音 |

---

## 四、主要用法：配「do-si-la-sol」下行旋律 ⭐

### 核心组合

```
旋律：  do    si    la    sol
和弦：  T    DTⅢ    S    D/T
      (或TSⅥ)      (极少TSⅥ)  (或K₄₆)
```

### 为什么这是 DTⅢ 的「主要用法」

``` 没学 DTⅢ 时 si→la 只有三种配法：
  ┌─ si 配 D/D₇，la 配 S/SⅡ → ❌ 逆功能
  ├─ si 配 D/D₇，la 配 TSⅥ → 阻碍进行感，不一定想要
  └─ si 配 DVII₇，la 配 DVII₇ → 都是属功能，单调

学了 DTⅢ 之后：
  ✅ si→DTⅢ（小三和弦），la→S（大三和弦）
  → 小三→大三 = 色彩从暗淡到明亮，新鲜感！
  → DTⅢ 复合了主功能，接 S 不算逆功能
```

> **la 优先配 S（大三和弦）而非 TSⅥ（小三和弦）**，为了小三→大三的色彩对比。

### DTⅢ 后面可以接什么

| 后面的 la 配 | 频率 | 原因 |
|:---:|:---:|------|
| **S** | ⭐⭐⭐ | 小三→大三，新鲜感对比感最强 |
| TSⅥ | ⭐ | 小三→小三，无色彩对比 |

---

## 五、次要用法：经过组合 DTⅢ → D³₄ → T

```
低音：      mi → re → do
上方三声部： sol fa mi   （可轮转）
            mi sol sol
            si si do
```

> 各声部横向 ≤ 三度，非常平稳。已加入经过/辅助和弦总结表。

---

## 六、补充：经过/辅助和弦的两种标准

以前要求「中间和弦不强于+不长于**两侧**」，本课修正放宽：

| 标准 | 要求 | 适用范围 |
|:---:|------|------|
| **严格**（旧）| 不强于+不长于**两侧** | 最优，确保经过感 |
| **宽松**（新）| 不强于+不长于**前面**即可 | 无法满足严格版时的替代 |

### 哪些和弦只能用固定搭配（不能单独使用）

| 和弦 | 原因 |
|------|------|
| T₄₆、S₄₆、D₄₆、TSⅥ⁴₆ | 四六和弦，不能单独使用 |
| 重复三音的 T₆ | 功能不明确 |
| T → D³₄ → T₆ 里的 D³₄（七音上行）| 特殊进行 |

> 其他组合里的和弦如果本身可单独使用（如 SⅡ₂），不强于不长于也可当普通连接。

---

## 七、模式零 + 模式一用法总结

| 模式 | DTⅢ 用法 |
|------|------|
| **模式零** | 仅两个用法：① do-si-la-sol 旋律 ② DTⅢ→D³₄→T 经过组合。其他地方尽量不用，实在配不下偶尔用 |
| **模式一** | 旋律以 mi-sol-si 为主时可直接配 DTⅢ，不限制 |

## 八、调内七个和弦全部学完！

```
大调七个和弦（C大调）：

I   T     C-E-G    大三和弦  主功能
II  SⅡ    D-F-A    小三和弦  下属功能
III DTⅢ   E-G-B    小三和弦  属+主复合（偏属）
IV  S     F-A-C    大三和弦  下属功能
V   D     G-B-D    大三和弦  属功能
VI  TSⅥ   A-C-E    小三和弦  主+下属复合（偏主）
VII DVII  B-D-F    减三和弦  属功能（最不协和）
```

---

## 练习要求

- **模式零**：高音题 + 低音题各一道，大调题找 do-si-la-sol 旋律使用 DTⅢ（小调不用）
- **模式一**：一道旋律，柱式/分解/混合各配一个，旋律合适时可使用 DTⅢ
